## 四、计算题
**1. 直接写得数：**

**2. 简便计算：**
(1) 25×32×1.25 = 25×4×8×1.25 = 100×10 = 1000
(2) 36×99+36 = 36×(99+1) = 36×100 = 3600
(3) 3.28+2.76+6.72+7.24 = (3.28+6.72)+(2.76+7.24) = 10+10 = 20
(4) 125×88 = 125×8×11 = 1000×11 = 11000

## 六、解决问题
1. 方法一：4×24×1.5 = 96×1.5 = 144元
   方法二：4×1.5×24 = 6×24 = 144元
2. 底边 = 48-18×2 = 12cm；三边相等：48÷3 = 16cm
3. (1) (138+142+145+140+139+136)÷6 = 840÷6 = 140cm
   (2) (840+142)÷7 = 982÷7 ≈ 140.3cm
4. 假设全汽车：12×4=48轮，多48-40=8轮
   三轮：8÷(4-3)=8辆，汽车：12-8=4辆
5. (1) (1.2+0.8)-2.5 = 2-2.5 = -0.5km（路线A比路线B长0.5km）
   (2) 2+2.5 = 4.5km

= (10+100+1000+10000) - 0.4 + 0.4 = 11110
