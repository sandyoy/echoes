# 社区居家康复体验点 · 民政局合作方案

> 版本：v2.0（2026-06-25）
> 背景：居家康复元年政策窗口期 + 社区食堂/驿站模式升级
> 目标：与民政局合作，在社区食堂/长者饭堂/老年活动中心设立居家康复体验点

---

## 一、核心逻辑：为什么选民政局，不选卫健委？

| 路径 | 卫健委 | 民政局 |
|------|--------|--------|
| 管什么 | 治病、医疗 | 养老、社区服务 |
| 痛点 | 床位不够、康复科忙不过来 | 社区老年人健康管理手段有限 |
| 能给什么 | 开处方、转诊 | 场地、政策通道、补贴、长护险 |
| 圆爱康能对接什么 | 合作医院转诊 | **免费设体验点 + 落地居家康复服务** |

**核心判断：** 居家康复卡在"医疗"和"养老"中间。走民政局这条路更直接——场地、人流、政策通道都有了。医疗部分通过合作医院解决。

---

## 二、对民政局的价值（你要去谈时的说辞）

民政局现在面临的问题：

1. **社区食堂/养老驿站利用率低** — 吃完饭人就走了，下午场地空着
2. **老年人健康管理手段不够** — 光搞讲座没人来，没抓手
3. **居家养老服务没产品** — 长护险在推，但没有好的居家康复服务包可落地
4. **需要政绩亮点** — "基层医养结合示范点"是个好招牌

**圆爱康的解决方案：**

> 免费在社区食堂/驿站设一个"居家康复体验角"，圆爱康出设备、出人、出运营。民政局只提供场地。
>
> 成果：每个点每月服务XXX人次、建立健康档案、筛选有康复需求的长者对接专业服务。
>
> 成本：零。圆爱康承担全部设备和人力成本。

---

## 三、方案框架：从体验到服务的完整链路

```
社区食堂/驿站（民政局协调场地）
       ↓
  免费体验角（圆爱康出设备+人员）
  ├── 圆小汫（生物反馈电刺激治疗仪）免费体验
  ├── 体态评估（10分钟出报告）
  ├── 血压血糖检测
  ├── 专注力/认知趣味测评
  ├── 健康讲座（每周1-2次）
       ↓
  扫码 → 企业微信（加好友）
       ↓
  线上持续触达
  ├── 健康科普（每周1-2篇）
  ├── 康复知识分享
  ├── 活动通知
  ├── 长护险政策解读
       ↓
  有需求时 → 用户主动咨询
       ↓
  居家康复服务交付
  ├── 设备租赁/购买（圆小汫等）
  ├── 六师共管远程指导
  ├── 康复计划制定+跟踪
  └── 复诊评估对接合作医院
```

## 五、三个层次（分阶段落地）

### 第一层：免费体验点（1-2个月落地）

**目标：** 建立线下触点，积累用户，验证转化率

**配置（一个点的标准）：**

| 项目 | 内容 | 成本 |
|------|------|------|
| 场地 | 社区食堂下午14:00-17:00 | 免费（民政局协调） |
| 设备 | 圆小汫1台 + 血压计1台 + 体态评估平板 | 圆爱康现有设备 |
| 人员 | 1名兼职人员（可招社区志愿者） | 3,000-4,000元/月 |
| 物料 | 易拉宝、宣传页、二维码桌牌 | 设计+打印约500元 |
| 线上 | 企业微信、H5页面（已有） | 已有 |

**体验内容：**
- 圆小汫免费体验（10-15分钟/人次）
- 体态评估（10分钟出报告）
- 血压血糖检测
- 专注力/认知趣味测评

**关键指标：**
- 每日到场人数
- 扫码加企微率（目标：>50%）
- 体验后主动咨询率（目标：>10%）
- 最终转化率（目标：>3%）

### 第二层：健康驿站（3-6个月，跑通后复制）

**目标：** 形成标准化模式，复制到3-5个点

在第一层基础上增加：
- 每周1-2次健康讲座/康复知识分享
- 每月1次三甲医院专家义诊（合作医院医生）
- 建立社区健康档案
- 对接长护险申请通道

### 第三层：社区康复服务站（6-12个月）

**目标：** 成为民政局认可的社区康复服务定点单位

- 设备可租可售（圆小汫居家版）
- 六师共管线上跟进
- 与合作医院打通转诊通道
- 纳入长护险报销范围

## 六、民政局谈判要点

### 你来谈的时候，这样说：

**开门见山：**
> "我们是圆爱康生物科技，做居家康复的。想在社区食堂免费设个体验角，不占地方、不收费、您零成本。"

**民政局会问：**
1. **"你们是干什么的？"**
   → 做居家康复的。合作医院有珠海妇幼、顺德人民医院等。核心是六师共管（医生+治疗师+护士+客服+工程师+督导）远程指导患者在家做康复。

2. **"为什么要设体验点？"**
   → 社区很多老人有康复需求（腰腿疼、尿失禁、便秘、术后恢复），但不知道去哪做、也没人告诉他们可以做居家康复。免费体验让他们亲身感受，再决定要不要。

3. **"对民政局有什么好处？"**
   → ①填补社区居家康复服务空白 ②为长护险落地提供服务承接方 ③打造"社区医养结合"示范点 ④零成本、零风险

4. **"怎么确保合规？"**
   → 现场不收费、不推销、不收集敏感信息。用户扫码加的是企业微信，后续有需求时用户主动咨询，我们才推荐服务。现场只有公益健康服务。

### 合作备忘录格式：

```
甲方（民政局/社区服务中心）：提供场地 + 协调社区资源
乙方（圆爱康）：提供设备 + 运营人员 + 服务

合作内容：
1. 甲方提供XX社区食堂/驿站XX场地，每周X天、每天X小时
2. 乙方免费设置居家康复体验角，提供设备、运营、维护
3. 所有体验服务免费，不收取居民任何费用
4. 双方每季度评估一次效果，决定是否续约或扩展
```

## 八、风险与注意

⚠️ **合规红线：** 体验点现场绝对不能收费、不能推销。所有商业转化必须在线上完成、用户主动发起。

⚠️ **人力问题：** 每个点需要有人值守。初期可以用圆爱康现有人员，后期复制时需要招专职或社区志愿者。

⚠️ **效果不确定：** 体验点本质上是个引流漏斗，转化率需要实际跑才能知道。第一批目标不要高，跑通流程就是胜利。

⚠️ **民政局层级：** 区级民政局推得快，市级可能要更长时间。建议从区级民政局切入。
