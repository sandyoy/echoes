## 一、项目背景

### 已有资源
- ✅ **一家公司**：主营居家康复数字疗法（盆底康复、心脏术后康复、儿童ADHD数字疗法、体态评估）
- ✅ **三甲医院合作**：已与多家三甲医院签约，提供居家服务包（免费提供设备耗材，服务后分成）
- ✅ **家家健健康管理研究中心**：民政局注册的非营利组织
- ✅ **设备**：二类医疗器械评估设备（体态评估仪等，价值十几万）在仓库

### 核心目标
在社区找到免费场地做 **线下体验点**，引流并转化为 **线上居家服务**。

---

## 二、方向探索

| 方向 | 结论 | 原因 |
|------|:----:|------|
| 三甲医院旁边设体验点 | ❌ 否决 | 会与合作医院产生"抢患者"冲突 |
| 母婴店/产康机构合作 | ❌ 否决 | 双方诉求不一致，难合作 |
| 党群服务中心四点半学堂 | ⚠️ 可引流 | 儿童治疗依赖医生，转化路径太长 |
| **社区食堂 + 健康驿站** | ✅ **选定** | 免费场地+下午空闲时段 |

---

## 三、选定方案：社区食堂 + 健康驿站

### 用户路径

```
社区食堂（免费场地）
  ↓ 下午空闲时段驻点活动
健康角（免费评估）
  ↓ 体态评估、血压血糖、专注力测评
扫码进家家健 H5
  ↓ 双入口
┌──────────────────────────────┐
│ ① 公益活动                    │
│    免费讲座、义诊、科普         │
│    （家家健品牌·非营利）         │
├──────────────────────────────┤
│ ② 健康服务                    │
│    评估报告、健康咨询           │
│    （需用户主动授权）            │
└──────────────────────────────┘
  ↓
引导关注家家健公众号 + 企业微信
  ↓
长期触达 → 产生需求 → 导流到公司做居家服务
```

### 关键决策

| 决策事项 | 结论 |
|---------|------|
| **运营主体** | 家家健健康管理研究中心（非营利）做线上平台和活动 |
| **商业转化** | 你的公司（营利）做服务交付 |
| **公益/商业切割** | 线上线下严格分离，用户主动授权后才能推荐商业服务 |
| **场地来源** | 社区食堂（下午空闲时段） |
| **收费模式** | 现场全部免费，不在体验点收一分钱 |
| **用户路径** | 食堂扫码 → H5 → 关注公众号/企微 → 长期触达 → 有需求时转化 |
| **核心设备** | 体态评估仪、心理测评平板、血压血糖仪 |
