## 一、项目背景：基层慢病管理的现实困境

### 1.1 筛查做了，但"筛完就完"

当前基层医疗卫生机构每年承担大量慢病筛查任务（盆底功能筛查、心血管风险评估、认知功能初筛等），但普遍存在一个断层：**筛查结果出来后，缺乏后续的干预和管理闭环。**

- 患者拿到评估报告，医生告知"建议关注"，但后续怎么做、谁来管，没有明确路径
- 轻症患者得不到及时干预，拖到中重症才去三甲医院，增加医疗负担
- 筛查本身成为"任务导向"而非"健康导向"，投入产出比低

### 1.2 医院康复断在出院那一刻

二三级医院的康复科室能力很强，但患者出院后的居家康复几乎是空白：
- 心脏术后患者出院后缺乏持续监护，再住院率高
- 盆底患者完成院内治疗后，缺少家庭巩固训练的指导
- 认知障碍患者的训练需要长期坚持，医院无法持续跟踪

### 1.3 基层缺人手、缺服务能力

基层医疗机构面临"有场地、有设备、缺人管"的困境：
- 公共卫生服务人员编制有限，难以增加工作量
- 居家康复需要持续跟踪、在线指导、数据监测，传统模式难以承载
- 患者"去哪做、找谁做、怎么做"三个问题没有系统性解决方案

### 1.4 分级诊疗缺乏"居家"这一环

分级诊疗体系中有社区卫生服务中心、有二三级医院，但**从医院到家庭的"最后一公里"仍是空白**。居家康复不是单纯的"出院后自己注意"，而应该是有专业管理、有数据跟踪、有人负责的服务体系。

## 二、方案概述：三级分流+居家延伸

### 2.1 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                    社区卫生服务中心                          │
│                    （筛查发现 + 初步评估）                    │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
              ┌──────────────┐
              │  医生诊断分层  │
              └──┬───────┬───┘
                 │       │
        ┌────────▼──┐  ┌─▼───────────┐
        │  轻症/稳定期 │  │ 中重症/急性期 │
        └────────┬───┘  └─┬───────────┘
                 │         │
        ┌────────▼───┐  ┌─▼───────────┐
        │ 居家康复通道 │  │ 转诊二三级医院 │
        │（圆爱康承接）│  │ （院内治疗）  │
        └────────┬───┘  └──────┬──────┘
                 │              │
                 └──────┬───────┘
                        │
                 ┌──────▼───────┐
                 │  居家康复管理  │
                 │（六师共管跟踪）│
                 └──────────────┘
```

### 2.2 核心逻辑：筛得出、接得住、管得好

| 环节 | 做什么 | 谁负责 |
|:---|:---|:---:|
| **筛得出** | 利用现有医疗机构的人和设备，开展常态化筛查 | 社区/医院现有医护 |
| **接得住** | 筛查结果由医生诊断，分层后提供不同路径 | 医生 + 圆爱康培训赋能 |
| **管得好** | 居家康复全程跟踪，效果数据化 | 圆爱康六师共管体系 |

### 2.3 项目定位

本项目不是新建一套服务体系，而是在**现有医疗机构康复项目基础上，增加一条居家通道**。患者多一个选择，医护多一个工具，政府多一项民生服务。

## 三、病种服务体系

### 3.1 盆底康复（含便秘、尿失禁）

#### 3.1.1 现状与痛点

盆底功能障碍是女性高发问题（产后、更年期、老年群体），但就诊率不足30%：
- 很多患者"不好意思说"，认为漏尿、便秘是正常现象
- 知道有问题但不愿去医院做康复（时间成本、心理门槛）
- 院内盆底康复疗程结束后，需要长期家庭巩固训练，但缺乏专业指导
- 便秘和尿失禁在临床上都归入盆底范畴，但很多患者不知道盆底康复可以解决这些问题

#### 3.1.2 项目流程

```
社区/医院筛查（表面肌电评估）
        ↓
医生解读评估报告 + 诊断
        ↓
    ┌───┴───┐
    │       │
  轻症    中重症
    │       │
 居家康复  转诊/院内治疗
    │       │
    └───┬───┘
        ↓
  居家康复管理（六师共管）
（设备指导 + 训练计划 + 数据跟踪）
```

#### 3.1.3 设备清单

| 设备名称 | 功能简介 | 配置方式 | 备注 |
|:---|:---|:---:|:---|
| 盆底表面肌电评估仪 | 评估盆底肌力、耐力、协调性，出具评估报告 | ①医院已有 ②建议采购 | 筛查/评估用 |
| 盆底康复治疗仪（居家可用） | 患者带回家，按训练方案进行居家康复治疗 | 含在服务包内 | 可循环使用 |
| 盆底生物反馈治疗仪 | 院内治疗用，配合理疗 | 建议医院采购 | 部分医院已有 |

#### 3.1.4 服务内容

- **筛查评估：** 社区/医院现有医护操作表面肌电设备，系统自动生成评估报告
- **医生诊断：** 医生结合临床判断，决定患者进入居家康复通道或转诊
- **居家康复方案：** 根据评估结果定制个性化居家康复方案（训练频率、强度、周期）
- **六师共管跟踪：**
  - 康复治疗师在线指导设备使用和动作纠正
  - 健康管理师定期回访，跟踪训练依从性
  - 护士解答日常疑问
  - 营养师提供饮食建议（便秘管理）
  - 心理支持（长期症状患者的情绪管理）
  - 运动指导师提供辅助运动方案
- **数据闭环：** 居家训练数据实时上传，管理者可查看完成情况和效果趋势

#### 3.1.5 结算模式

- **服务包形式：** 按例结算（每人次康复周期）
- 包含：居家设备使用 + 六师共管服务 + 数据跟踪
- **费用流转方式见第四章**

### 3.2 心脏康复

#### 3.2.1 现状与痛点

心脏康复（心梗术后、支架术后、心衰稳定期）是循证医学I类推荐，但实际参与率极低：
- 全国心脏康复参与率不足10%
- 主要原因：患者出院后回社区，无专业机构承接；每周2-3次去医院做康复不现实
- 居家心脏康复在国际上已有成熟模式（英国NICE指南、美国AHA均推荐），国内尚在起步
- 心脏康复是"时间窗口期"——出院后3个月内是黄金康复期，错过效果打折扣

#### 3.2.2 项目流程

```
院内评估（心超/CPET/6分钟步行试验等，在医院完成）
        ↓
医生诊断 + 制定居家康复方案
        ↓
    ┌───┴───┐
    │       │
  低危    中高危
    │       │
 居家康复  院内康复+居家过渡
    │       │
    └───┬───┘
        ↓
  居家心脏康复管理
（远程心电监护 + 运动处方 + 六师共管）
```

#### 3.2.3 设备清单

| 设备名称 | 功能简介 | 配置方式 | 备注 |
|:---|:---|:---:|:---|
| 心脏超声/CPET | 心功能评估（射血分数、运动耐量等） | 医院已有 | 常规设备，三甲/二级均有 |
| 6分钟步行试验工具 | 简易心功能评估 | 医院已有 | 社区可操作 |
| 居家心电监测设备 | 患者在家佩戴，实时监测心率、心律 | 含在服务包内 | 可循环使用 |
| 居家康复设备 | 超声电治疗康复设备（居家可用） | 含在服务包内 | 循环使用 |

#### 3.2.4 服务内容

- **院内评估：** 心超/CPET/6分钟步行试验由医院完成（现有能力，无需新增）
- **居家康复方案：** 心脏康复医师出具运动处方（频率、强度、时间、类型）
- **远程监护：** 患者居家训练时佩戴心电设备，数据实时上传，异常自动预警
- **六师共管跟踪：**
  - 心脏康复治疗师在线调整运动处方
  - 护士监测生命体征数据
  - 营养师提供心脏健康饮食方案
  - 心理支持（术后焦虑/抑郁管理）
  - 健康管理师跟踪服药依从性
- **应急预案：** 异常数据实时预警，触发医护响应机制

#### 3.2.5 结算模式

同3.1.5——按例结算，费用流转见第四章。

### 3.3 心理/认知康复（含阿尔茨海默症、ADHD）

#### 3.3.1 现状与痛点

心理和认知健康是基层医疗最薄弱的环节之一：
- 阿尔茨海默症早期识别率极低，多数患者就诊时已是中晚期
- 儿童ADHD：家长认知不足，学校发现难，社区缺乏评估能力
- 轻中度抑郁/焦虑：基层开药多、非药物干预少
- 认知训练需要"高频、长期、有趣"，传统门诊模式无法满足
- 数字疗法在认知康复领域已有大量循证证据，但基层尚未引入

#### 3.3.2 项目流程

```
社区/学校/家庭 → 初筛（简易认知/行为量表）
        ↓
医院评估（专业量表 + 临床访谈）
        ↓
医生诊断 + 分层
        ↓
    ┌───┴───┐
    │       │
  轻中度   中重度
    │       │
 居家数字疗法 院内治疗+药物+居家辅助
    │       │
    └───┬───┘
        ↓
  居家康复管理（数字疗法 + 人文关怀 + 六师共管）
```

#### 3.3.3 设备清单

| 设备名称 | 功能简介 | 配置方式 | 备注 |
|:---|:---|:---:|:---|
| 认知评估量表（数字化） | 简易精神状态检查、蒙特利尔认知评估等 | 第三方提供 | 软件形式，无需硬件 |
| ADHD评估量表（数字化） | 家长问卷、教师问卷、行为观察 | 第三方提供 | 软件形式 |
| 经颅电刺激仪 | 院内治疗用（抑郁/认知障碍） | 建议医院采购 | 院内治疗设备 |
| 数字疗法终端 | 患者在家完成认知训练任务 | 含在服务包内 | 手机/平板可用 |

**注：** 心理/认知方向是 **"评估+居家闭环"** 的模式——评估（第三方提供数字化量表）+ 居家康复（数字疗法 + 六师共管）由圆爱康完成；院内治疗（经颅电刺激等）由医院自行配置。

#### 3.3.4 服务内容

- **初筛：** 社区/学校使用简易数字化量表进行人群初筛（5-10分钟）
- **评估：** 医疗机构进行专业评估（临床访谈 + 完整量表）
- **居家数字疗法：**
  - 认知训练游戏化任务（针对阿尔茨海默症：记忆力、注意力、执行功能训练）
  - ADHD儿童执行功能训练（行为管理、注意力训练）
  - 轻中度抑郁/焦虑的认知行为疗法数字化模块
- **六师共管跟踪：**
  - 心理治疗师在线指导训练
  - 健康管理师跟踪依从性
  - 家人/照顾者支持指导
  - 数据趋势分析，效果可视化

#### 3.3.5 结算模式

同3.1.5——按例结算，费用流转见第四章。

## 四、卫健局的资金投入与结算模式

本方案的核心优势在于：**卫健局按效果付费，钱花得明白、管得轻松。**

### 4.1 卫健局的资金投入方式

卫健局作为项目主导方，有三种投入方式可选：

| 投入方式 | 说明 | 适合场景 |
|:---|:---|:---|
| **按服务例数结算** | 卫健局按实际完成的筛查例数+康复例数，定期向服务方支付服务费 | 试点启动，零设备投入 |
| **设备采购+服务费** | 卫健局一次性采购核心评估设备（归卫健局/医院资产），康复服务费按例结算 | 有设备预算，想提升院内能力 |
| **全包干** | 卫健局按年度/周期向服务方打包采购全部服务 | 预算充足，想省心管理 |

### 4.2 卫健局的资金流向设计

```
卫健局专项经费
       │
       ▼
┌─────────────────────┐
│  卫健局监管账户      │ ← 卫健局指定，资金流向全程可查
│  （统一收付平台）     │
└──────────┬──────────┘
           │
    ┌──────┴──────┐
    ▼             ▼
服务提供方      医院/社区
（圆爱康）      （机构管理费）
    │
    ▼
居家康复服务执行
（设备+六师共管+数据）
```

**卫健局视角的五大优势：**

1. **钱不进医院大账** — 服务费通过卫健局监管账户直接结算，避免资金进入基层医院后难以流出的体制问题
2. **按效果付费，不做冤大头** — 卫健局只按实际完成的康复例数结算，做一例付一例，不做不付
3. **价格透明可审计** — 统一结算标准，全区一个价，卫健局随时可以审计
4. **零运营负担** — 卫健局只负责监督考核，不负责日常运营、人员管理、设备维护
5. **数据可追溯** — 每个患者的康复数据、服务记录都在平台上，卫健局可随时查看项目进展

### 4.3 卫健局的结算与考核标准

卫健局通过以下机制确保服务质量：

| 环节 | 卫健局的管控手段 |
|:---|:---|
| **服务准入** | 服务方需通过卫健局资质审核，备案服务方案和收费标准 |
| **过程监督** | 卫健局可随时调取康复数据平台，查看筛查人数、康复完成率、患者满意度 |
| **效果考核** | 按季度评估：康复完成率≥80%、患者满意度≥90%、再住院率下降等指标 |
| **结算条件** | 达到考核标准后，按例/按周期支付服务费；未达标可扣减或暂停 |
| **年度复审** | 每年重新评估服务方资质，优胜劣汰 |

### 4.4 结算价格参考

> 以下价格为预估参考，具体根据当地卫健局预算和项目规模商议确定。

| 服务类型 | 结算方式 | 参考价格 | 说明 |
|:---|:---|:---:|:---|
| 社区筛查（含评估报告） | 按例 | XX元/人次 | 含设备携带+人员+报告解读 |
| 居家康复服务包（全周期） | 按例 | XX元/人 | 含设备使用+六师共管+数据跟踪 |
| 年度全包服务 | 按年 | XX万元/年 | 含筛查+康复+培训+运营 |

## 五、采购与服务方案建议（灵活组合）

根据示范医院已有设备情况，提供三种灵活选择：

### 方案A：纯服务模式（推荐试点）

**适用场景：** 示范医院已有全部评估/治疗设备，仅需居家康复服务

| 采购内容 | 说明 | 配置方式 |
|:---|:---|:---:|
| 居家康复服务包 | 居家设备使用 + 六师共管 + 数据跟踪 | 按例采购 |
| 医护培训 | 居家康复理念培训 + 推荐话术培训 | 含在服务包内 |
| 患者教育材料 | 展板/易拉宝/宣传册/视频内容 | 第三方提供 |
| **卫健局投入** | **无需设备采购，按实际服务例数结算** | **零启动成本** |

### 方案B：核心设备+服务模式

**适用场景：** 医院有部分设备，需补充核心筛查/评估设备

| 采购内容 | 说明 | 配置方式 |
|:---|:---|:---:|
| 盆底表面肌电评估仪 | X台 | 卫健局采购 |
| 居家康复服务包 | 按例 | 按例采购 |
| 数字疗法终端 | 按病种 | 含在服务包内 |
| 医护培训+患者教育材料 | 全套 | 第三方提供 |
| **参考预算** | **设备约XX万 + 服务费按例** | |

### 方案C：全体系建设模式

## 八、关于圆爱康

### 8.1 公司定位

广州圆爱康生物科技有限公司是专注于**居家康复服务**的第三方服务机构，不派人驻场、不替代医疗机构职能，而是通过"培训赋能+六师共管+数据平台"的方式，为医院现有的康复项目增加居家通道。

### 8.2 核心能力

- **多病种覆盖：** 盆底康复（含便秘/尿失禁）、心脏康复、心理/认知康复（含阿尔茨海默症、儿童ADHD）、骨科康复等
- **六师共管体系：** 康复治疗师、健康管理师、护士、营养师、心理支持、运动指导师六位一体的居家管理
- **医院合作经验：** 已在多家三甲医院落地居家康复服务，跑通"培训-转化-服务-结算"全流程
- **数据化运营：** 居家康复数据实时上传，效果可量化、可追溯

### 8.3 合作理念

圆爱康坚持 **"赋能而非替代"** 的合作理念：
- 利用医院现有的培训时间、现有的人员、现有的项目基础
- 给患者多一个选择，给医护多一个工具，给政府多一项民生服务
- 做医院康复能力的延伸，而不是另建一套体系
