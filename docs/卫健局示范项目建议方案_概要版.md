# XX区「筛查+居家康复」慢病管理示范项目（概要版）

> **一句话定位：** 给医院已有的康复项目加一条居家通道——不新建、不派人、不分成，让患者多一个选择，让医护多一个工具。

---

## 一、为什么需要这个项目？

**三个数字说明问题：**

| 问题 | 现状 |
|:---|:---|
| 筛查之后呢？ | 筛查做了，报告出了，但后续没人管——"筛完就完" |
| 出院之后呢？ | 患者在医院做完康复回家就断了——复发率高 |
| 居家谁来做？ | 基层缺人手、缺服务能力——居家康复是空白 |

**核心矛盾：** 分级诊疗喊了很多年，但"从医院到家庭"的最后一公里没人走通。

---

## 二、这个项目做什么？

**一个字：接得住。**

```
社区筛查 → 医生诊断 → 轻症走居家通道（圆爱康承接）
                      → 中重症走转诊通道（现有体系不变）
```

**项目不是新建服务体系**，而是在医疗机构已有的康复项目上，增加一条居家康复通道。患者在医院治疗期间就知道有居家选项，出院后无缝衔接。

---

## 三、覆盖哪些病种？

| 病种 | 评估方式 | 居家做什么 | 现状基础 |
|:---|:---|:---|:---:|
| 🏥 盆底康复（含便秘/尿失禁） | 表面肌电评估 | 居家训练+六师共管 | 最成熟，多家医院已落地 |
| ❤️ 心脏康复 | 医院评估（心超/CPET） | 远程监护+运动处方 | 需求大，国际已证明有效 |
| 🧠 心理/认知（含阿尔茨海默、ADHD） | 数字化量表评估 | 数字疗法+认知训练 | 基层最薄弱，增量空间大 |

**原则：** 优先选医院已有的项目（有存量基础），不做从零教育的项目。

---

## 四、圆爱康做什么？医院做什么？

**圆爱康的角色：赋能不是替代**

| 环节 | 谁来做 | 怎么配合 |
|:---|:---|:---|
| 筛查评估 | 👨‍⚕️ 医院现有医护 | 利用医院已有培训时间，植入居家康复理念 |
| 诊断分层 | 👨‍⚕️ 医生 | 给患者多一个选项：居家康复 ✅ |
| 居家服务 | 🏥 圆爱康六师共管 | 在线指导+数据跟踪，患者不用跑医院 |
| 费用结算 | 🏥 圆爱康统一收 | 按月向医护结算劳务——钱不进医院大账，避免出不来 |

---

## 五、钱怎么转？（关键设计）

```
卫健局/患者 → 圆爱康（统一收服务费）
     ↓
圆爱康 → 医护劳务（月结，按例计算）
圆爱康 → 医院管理费（按比例）
```

**为什么这样设计？**
- 钱不进医院大账 → 出得来、结得快
- 医护做一例结算一例 → 动力够、积极性高
- 卫健局按服务效果付费 → 钱花得值、可量化

## 六、三个方案，丰俭由人

| 方案 | 卫健局投入 | 适合 |
|:---|:---|:---|
| **A. 纯服务** | 零设备采购，按例付服务费 | 医院已有设备，先试点 |
| **B. 核心设备+服务** | 买关键筛查设备 + 按例买服务 | 2-3个病种稳步推进 |
| **C. 全体系建设** | 全套设备 + 全病种服务 | 预算充裕，打造标杆 |
