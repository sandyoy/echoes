# XX区「筛查评估+居家康复」慢病管理示范项目
## 专项债资金路径方案（概要版）

> **一句话定位：** 给医院已有的康复项目加一条居家通道——不新建、不派人、不分成，让患者多一个选择，让医护多一个工具。

---

## 一、为什么需要这个项目？

**三个数字说明问题：**

| 问题 | 现状 |
|:---|:---|
| 筛查之后呢？ | 筛查做了，报告出了，但后续没人管——"筛完就完" |
| 出院之后呢？ | 患者在医院做完康复回家就断了——复发率高 |
| 居家谁来做？ | 基层缺人手、缺服务能力——居家康复是空白 |

**核心矛盾：** 分级诊疗喊了很多年，但"从医院到家庭"的最后一公里没人走通。

---

## 二、这个项目做什么？

**一个字：接得住。**

```
社区筛查 → 医生诊断 → 轻症走居家通道（圆爱康承接）
                      → 中重症走转诊通道（现有体系不变）
```

**项目不是新建服务体系**，而是在医疗机构已有的康复项目上，增加一条居家康复通道。患者在医院治疗期间就知道有居家选项，出院后无缝衔接。

---

## 三、覆盖哪些病种？

| 病种 | 评估方式 | 居家做什么 | 现状基础 |
|:---|:---|:---|:---:|
| 🏥 盆底康复（含便秘/尿失禁） | 表面肌电评估 | 居家康复治疗仪+六师共管 | 最成熟，多家医院已落地 |
| ❤️ 心脏康复 | 医院评估（心超/CPET） | 超声电治疗康复设备+远程指导 | 需求大，国际已证明有效 |
| 🧠 心理/认知（含阿尔茨海默、ADHD） | 数字化量表评估 | 数字疗法+认知训练 | 基层最薄弱，增量空间大 |

**原则：** 优先选医院已有的项目（有存量基础），不做从零教育的项目。

---

## 四、圆爱康做什么？医院做什么？

**圆爱康的角色：赋能不是替代**

| 环节 | 谁来做 | 怎么配合 |
|:---|:---|:---|
| 筛查评估 | 👨‍⚕️ 医院现有医护 | 利用医院已有培训时间，植入居家康复理念 |
| 诊断分层 | 👨‍⚕️ 医生 | 给患者多一个选项：居家康复 ✅ |
| 居家服务 | 🏥 圆爱康六师共管 | 在线指导+数据跟踪，患者不用跑医院 |
| 费用结算 | 🏥 圆爱康统一收 | 按月向医护结算劳务——钱不进医院大账，避免出不来 |

---

## 五、卫健局的钱怎么花？（专项债资金路径）

```
              专项债（卫健局申请）
                    │
                    ▼
         ┌────────────────────────┐
         │    专项债资金池        │
         └────────┬───────────────┘
                  │
          ┌───────┴────────┐
          ▼                 ▼
  ┌─────────────┐   ┌──────────────┐
  │ ① 购买设备   │   │ ② 购买服务    │
  │ 一次性采购    │   │ 按例/按周期结算 │
  │ 资产归卫健局  │   │ 圆爱康提供服务  │
  └─────────────┘   └──────────────┘
```

**核心创新：两条资金线，设备+服务分开走**
- **设备线**（一次性）：专项债→买设备→配置到社区/医院
- **服务线**（持续性）：专项债→按效果买服务→患者享受居家康复

**给卫健局的好处：**
- ✅ **按效果付费** — 做一例付一例，不做不付
- ✅ **专款专用** — 不占当年预算指标，流程简化
- ✅ **设备+服务捆绑** — 买了就用、用了见效，不闲置
- ✅ **钱不走医院大账** — 通过专户直付，避免基层钱出不来
- ✅ **数据可追溯** — 每个患者的康复进度，卫健局随时查看
