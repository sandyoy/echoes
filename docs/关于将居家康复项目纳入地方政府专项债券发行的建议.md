## 一、背景：为什么在这个时间点提这个建议？

### 1.1 基层慢病管理面临的钱的问题

当前基层慢病管理的核心矛盾不是"不想做"，而是"没钱做"：

- **公卫经费有限**：基本公卫服务项目经费已涵盖大量常规工作，难以腾出资金启动新的服务项目
- **年度预算约束**：居家康复服务体系建设需要一次性设备投入 + 持续性服务采购，纯靠当年财政预算难以支撑
- **区级财政压力大**：基层财政"保工资、保运转"压力大，新增民生项目排期漫长

但没有投入就没有服务，患者等不起。

### 1.2 专项债是当前最合适的资金工具

2025年国务院办公厅出台《关于优化完善地方政府专项债券管理机制的意见》，明确将"卫生健康"纳入专项债支持范围。同时专项债具有以下特点：

- **不占当年预算指标**：发债资金专款专用，不挤占常规财政预算
- **期限长、利率低**：专项债期限可达10-20年，利率远低于市场融资
- **以项目收益偿还**：有经营性收益的项目才适合发债——居家康复服务恰好符合

**核心观点：** 居家康复项目天然适合用专项债资金启动——有设备投入（一次性）也有服务现金流（持续性），收益模式清晰，是专项债的优质标的。

### 1.3 圆爱康提出此建议的动机

圆爱康作为居家康复服务提供商，提出此建议并非推销单一产品，而是希望协助卫健局找到一个**可持续的资金模式**，让"社区筛查+居家康复"的闭环真正落地。卫健委和卫健局有项目、有政策、有需求，缺的是一套**能说服财政局的资金逻辑**——这个，圆爱康可以帮忙。

## 二、为什么居家康复项目适合申请专项债？

### 2.1 投向领域匹配

| 专项债支持领域 | 居家康复项目的对应点 |
|:---|:---|
| 卫生健康 | 基层慢病管理、康复医疗服务体系建设 |
| 养老托育 | 老年认知康复、盆底康复（中老年女性） |
| 公共卫生设施 | 社区筛查站点建设、设备配置 |
| 数字经济 | 康复数据平台、数字化评估系统 |

居家康复项目横跨多个专项债支持领域，申报立项有充分的政策依据。

### 2.2 卫健局的收益在哪里？

先搞清楚一笔账的流向，才能说清楚专项债怎么还。

**专项债的本质：** 卫健局发债借钱→花钱买设备+买服务→这些服务本身能产生收费收入→这个收入进入财政（非税收入）→用来还债。

所以卫健局的收益**不是**"卖设备赚多少钱"，而是**服务收费中进入财政的那一部分**。

**举例说明：**

假设一个居家康复服务包定价 **2,200元/人**，资金流向是这样的：

```
                    2,200元/人（患者或医保支付）
                            │
                  ┌─────────┴─────────┐
                  ▼                   ▼
      ┌────────────────┐  ┌──────────────────┐
      │ 圆爱康收取1,100 │  │ 卫健局非税收入1,100│
      │（设备使用+     │  │ （进入财政专户，   │
      │  六师共管服务） │  │  用于还债）      │
      └────────────────┘  └──────────────────┘
```

- **圆爱康收的部分**：设备提供费 + 六师共管服务费 + 数据跟踪
- **卫健局收的部分**：服务管理费（非税收入，进入财政专户，成为债券还款来源）

**其他服务同理：**
- 贴片电刺激服务包1,600元/人 → 圆爱康收800元 + **卫健局非税收入800元**
- 社区筛查评估200元/人次 → 圆爱康收100元 + **卫健局非税收入100元**

> **关键区别：** 专项债资金花的不是卫健局的"收益"，发债的钱是**借来的、要还的**。卫健局的还款来源是**服务运营中产生的非税收入**——每服务一个患者，卫健局就有一笔钱进财政账。服务量越大，还款能力越强。

**③ 债券还本付息能力**

发债500万，期限10年，年利率3%：

| 年度 | 还本（万元） | 付息（万元） | 本息合计（万元） | 卫健局非税收入（万元） | 偿债覆盖率 |
|:---|:---:|:---:|:---:|:---:|:---:|
| 第1-3年 | 0 | 45（3年合计） | 45 | 321（3年合计） | 713% |
| 第4-10年 | 500 | 68（7年合计） | 568 | 749（7年合计） | 132% |
| **总计** | **500** | **113** | **613** | **1,070** | **175%** |

**核心数据：** 卫健局年度非税收入约**107万元**。偿债能力分析：

- 前3年只付息不还本：年付息15万，非税收入107万 → 轻松覆盖（覆盖率713%）
- 第4年起分期还本：年均还本约71万+付息约10万=81万，非税收入107万 → 覆盖率约132%
- 第10年大额还本（120万）：覆盖率约87%，但前9年结余（每年107-81=26万×9年=234万）可覆盖

> **注意：** 这里的"项目收益"是卫健局层面的非税收入（107万/年），不是服务总价214万。设备采购是债务支出不是收益。设备是资产入账，不是现金流收入。

> 以上为示意测算，具体规模根据卫健局实际情况调整。

## 三、项目包装建议：居家康复如何"翻译"成专项债语言

专项债评审专家不是医疗专家，他们看的是："这个项目是什么？钱花在哪？收益从哪里来？怎么还？"

以下建议将居家康复项目"翻译"成评审专家看得懂的语言。

### 3.1 项目名称建议

> ❌ 内部名称：居家康复服务项目
> ✅ 申报名称：**XX区基层慢病"筛查-居家康复"服务体系建设项目**

**理由：** 突出"体系建设"而非"购买服务"，体现专项债的"建设"属性。

### 3.2 建设内容建议（三大模块）

#### 模块一：社区筛查评估能力建设（设备采购类）

在社区卫生服务中心配置筛查评估设备，建立"基层能筛查、筛查有结果、结果有跟踪"的常态机制。

- 盆底表面肌电评估仪（20台，社区配置）
- 数字化心理/认知评估系统（软件平台）
- 便携心功能评估辅助设备

#### 模块二：居家康复服务网络建设（服务采购类）

建立覆盖全区的居家康复服务网络，患者在家完成康复训练，专业团队远程管理。

- 居家康复设备（可循环使用）
- 六师共管服务团队（远程运营）
- 康复数据管理平台

#### 模块三：项目管理与能力建设（配套类）

- 医护培训（居家康复理念+操作）
- 患者教育体系
- 服务质量监管

### 3.3 收益来源建议（回答"钱从哪来"）

专项债最核心的问题：**收益在哪？**

建议卫健局将以下收入形式纳入项目收益测算：

| 收益来源 | 说明 | 性质 |
|:---|:---|:---|
| **设备集中采购价差收益** | 圆爱康按政府采购价（低于市场价30-40%）供应设备，卫健局按市场价入账 | 一次性 |
| **居家康复服务非税收入** | 居家康复服务费由卫健局统筹管理，作为非税收入入账，再按效拨付给服务提供方 | 持续性 |
| **数字化系统使用费** | 康复数据平台向医疗机构收取年度使用费 | 持续性 |
| **其他（可选）** | 健康宣教、增值服务等 | 辅助性 |

**操作建议：** 在项目可研报告中，将服务费收入纳入"专项收入"或"非税收入"科目，作为债券还款来源。

## 四、发债方案建议

### 4.1 发债规模

建议首期发债规模控制在 **500万-800万元**，理由如下：

- 设备采购约300万元（一次性）
- 服务费启动资金（预拨部分）约200-300万元
- 预留弹性空间

**小规模发债的优势：** 收益覆盖倍数高、审批通过率高、还款压力小。

### 4.2 债券期限

建议 **10年期**，匹配设备使用寿命（5-8年）+ 服务周期：

- 前3年：项目建设+运营
- 第4-10年：常态化运营，用服务收益还本付息
- 每年还本约50-80万（按500万规模），服务年收入约214万，完全覆盖

## 五、圆爱康能做什么？

圆爱康在专项债申报全流程中提供专业支持：

### 5.1 项目可研阶段

| 工作事项 | 圆爱康能提供的支持 |
|:---|:---|
| 项目建设内容撰写 | 提供居家康复服务体系的详细建设方案 |
| 设备清单与报价 | 提供政府采购价设备清单（低于市场价30-40%） |
| 收益测算 | 提供服务定价依据和收益测算模型 |
| 案例参考 | 提供已有合作医院的运营数据做案例支撑 |

### 5.2 申报材料阶段

- 配合编制项目可行性研究报告（可研报告）
- 提供居家康复服务的专业描述和行业数据
- 协助撰写"项目收益与融资自求平衡"章节

### 5.3 获批后落地

- 按PPP或服务采购协议，承接居家康复服务实际运营
- 承担从设备交付到六师共管的全链条服务
- 按月/按季度提供运营数据报告，支撑还本付息数据

## 六、下一步建议

如果卫健局认为此方向可行，建议按以下步骤推进：

1. **内部沟通**：卫健局内部讨论是否将居家康复专项债纳入年度计划
2. **前期对接**：与区财政局沟通专项债申报窗口期和额度
3. **可研编制**：委托专业机构编制项目可行性研究报告（圆爱康提供专业资料支持）
4. **申报入库**：在财政部的"地方政府专项债项目库"中申报入库
5. **发债落地**：获批后启动发债，资金到位后启动项目建设
