# 📖 拼音闪电卡 · 10天集中突破计划

**适合：** 4年级拼音薄弱的小朋友
**每天用时：** 10-15分钟
**方法：** 大声读 + 手指指 + 记口诀

## 📅 第1天：声母（上）b p m f d t n l

### 🔤 今日拼音
```
b   玻    像广播天线→肚子在右边
p   坡    泼水泼出去
m   摸    两扇门
f   佛    像根拐杖

d   得    像大肚子→肚子在左边 ✨易混
t   特    小伞把
n   讷    嘴巴闭一下才出声  ✨易混
l   勒    舌头抵上颚，气流走两边  ✨易混
```

### ✏️ 读一读
```
bā bá bǎ bà    pā pá pǎ pà    mā má mǎ mà
dā dá dǎ dà    tā tá tǎ tà    nā ná nǎ nà
lā lá lǎ là
```

### 🎯 今日口诀
```
b和d，像对双胞胎
b的肚子向右拐 → 📻（广播天线）
d的肚子向左歪 → 🐻（大肚子熊）

n和l，舌头位置不一样
n：鼻子出气，嘴巴闭
l：舌头卷，气流从两边飞
```

### ✅ 完成打勾
□ 每个读3遍
□ 易混对比读（b-d, n-l）
□ 选3个字造句

## 📅 第3天：平舌 vs 翘舌（z c s vs zh ch sh r）

### 🔤 今日拼音
```
z   资    平舌，舌尖抵下牙  ✨易混
c   次    平舌+吐气  ✨易混
s   丝    平舌，蛇吐信子  ✨易混
---
zh  知    翘舌，舌尖卷上去  ✨易混
ch  吃    翘舌+用力吐气  ✨易混
sh  诗    翘舌，气流从缝里  ✨易混
r   日    翘舌，声带振动
```

### ✏️ 对比读
```
zā zá zǎ zà  —  zhā zhá zhǎ zhà
cā cā cā cā  —  chā chá chǎ chà
sā sá sǎ sà  —  shā shá shǎ shà
```

### 🎯 今日口诀
```
平舌（z c s）：舌头放平像滑梯
翘舌（zh ch sh r）：舌尖卷起像勺子

四（sì）是（shì）
四不是十，十不是四
四是sì，十是shí
```

### ✅ 完成打勾
□ 平翘舌对比各读3遍
□ 练习区分：四是sì，十是shí
□ 读绕口令


## 📅 第9天：声调专项

### 🎵 四声练习
```
第一声（¯）高平：mā 妈   像在走平路
第二声（ˊ）上扬：má 麻   像在上坡
第三声（ˇ）拐弯：mǎ 马   像在下坡又上坡✨最易错
第四声（ˋ）下降：mà 骂   像在下坡
```

### ✏️ 易混声调练习
```
第二声 vs 第三声（最容易混！）：
yú（鱼）— yǔ（雨）
rén（人）— rěn（忍）
qí（旗）— qǐ（起）

第三声变调规则：
两个三声连读，第一个变二声
ní hǎo（你好）= 不是 nǐ hǎo，是 ní hǎo
```

### 🎯 今日秘籍
**第三声技巧：**
```
第三声最难读，记住一句话：
"先往下降，再往上升"
mǎ = 先到最低，再起来

练习三个字：
你（nǐ）→ 先降后升
我（wǒ）→ 先降后升
好（hǎo）→ 先降后升
```

### ✅ 完成打勾
□ 四声练习：mā má mǎ mà 读顺
□ 重点：二声 vs 三声对比读
□ 读出"你好"的正确变调
