# 基层社区慢病"筛查-干预-康复"三级管理体系整体建设方案

**编制单位：圆爱康健康科技（第三方服务机构）**

---

## 一、背景：基层慢病管理的现实困境

当前基层社区慢病管理面临四个痛点：

**痛点一：公卫考核指标压力大**
筛查覆盖率、慢病管理率、康复随访率是基层的硬指标。但社区人手有限，既要完成日常门诊，又要做慢病管理，往往心有余力不足。

**痛点二：设备买了，人跟不上**
很多社区采购了评估设备，但缺少操作人员，设备长期闲置。买设备容易，养人难。

**痛点三：筛查完就断了**
筛出阳性患者后，缺乏有效的转诊和持续管理机制。很多患者被告知"有问题"之后，就没有了下文——不重视、没时间、不知道去哪里做康复。

**痛点四：院外康复是空白**
患者出院或完成手术回家后，基本处于"无人管"状态。心衰术后3-6个月的黄金康复期、产后盆底修复窗口期、认知功能障碍的早期干预期，都在家中被浪费了。复发率高、再住院率高，最终成本还是落在医保上。

> **核心问题：设备好买，服务难建。基层缺的不是设备，是持续、专业的慢病服务能力。**

---

## 二、方案概述

### 2.1 项目定位

由第三方服务机构（圆爱康）为基层社区提供 **"筛查设备+专业人员+居家康复服务"一体化整体解决方案**，帮助卫健局实现：

- ✅ **不用养人**——筛查执行、康复指导由第三方专业人员完成
- ✅ **设备不闲置**——评估设备灵活配置，居家康复设备循环使用
- ✅ **完成公卫指标**——筛查覆盖率、慢病管理率、随访率数据可追踪
- ✅ **费用可控**——按服务效果付费，做一例结一例

### 2.2 服务对象

- **盆底健康**：产后女性、盆底功能障碍患者（中老年女性为主）
- **心脏康复**：冠心病术后、心衰稳定期、PCI术后患者
- **心理健康**：轻度认知障碍（MCI）、阿尔兹海默症早期、ADHD儿童及成人

### 2.3 服务模式（三级分流机制）

```
社区健康角/筛查日
       │
       ▼
   ┌────────────┐
   │  筛查评估   │  ← 圆爱康携带设备+人员进社区
   └──────┬─────┘
          │
    ┌─────┴─────┐
    ▼           ▼
  阴性        阳性
（健康宣教）    │
          ┌────┴────┐
          ▼         ▼
       轻症      中重症
    （居家康复）  （转诊二三级医院）
          │         │
          ▼         ▼
    ┌─────────┐  ┌─────────┐
    │设备回家  │  │院内治疗后│
    │远程指导  │  │转居家康复│
    │六师共管  │  └─────────┘
    └─────────┘
          │
          ▼
   效果评估·完成康复
```

## 三、三大慢病服务体系

---

### 3.1 盆底健康服务体系

#### 3.1.1 现状与痛点

盆底功能障碍是女性高发问题，产后发生率达30-40%，中老年女性发生率更是超过50%。然而大众认知度低，主动筛查率极低。即便筛查出问题的患者，也常因"去医院麻烦""不知道去哪做康复"而放弃治疗。

#### 3.1.2 项目流程

```
① 社区健康角/产后筛查日
      ↓ 圆爱康携带表面肌电评估仪进社区
② 快速评估（15-20分钟/人）
      ↓
③ 阳性 → 专人解读报告、告知治疗方案
      ↓
④ 患者选择居家康复方案 → 签订服务协议 → 领取设备
      ↓
⑤ 居家康复（圆爱康六师共管远程指导）
      ↓
⑥ 周期结束 → 效果复评 → 康复完成
```

#### 3.1.3 所需设备清单

| 环节 | 设备名称 | 参考价格 | 配置方式 |
|:---|:---|:---:|:---|
| **评估** | 盆底表面肌电评估仪 | 8-25万 | **由第三方服务机构提供，无需采购**（圆爱康自产设备） |
| **院内治疗** | 盆底康复治疗仪 | 10-30万 | 可由第三方服务机构整合提供，或医院自行采购 |
| **居家康复** | 居家盆底康复训练器（家用型） | — | **含在服务费内**，患者借用/租赁，循环使用 |

> ✅ 盆底线评估设备由圆爱康自带，医院/社区无需采购即可开展项目

#### 3.1.4 服务内容

- 社区/产后筛查日运营执行
- 患者管理档案建立
- 居家康复方案定制
- 六师共管（康复师+营养师+运动处方师+心理咨询师+健康管理师+药师）
- 数据追踪与效果评估

#### 3.1.5 结算模式

按完成康复例数结算，每月/每季度汇总。

---

### 3.2 心脏康复服务体系

#### 3.2.1 现状与痛点

心脏康复是心血管疾病全程管理中的关键环节。大量临床研究证实，科学的心脏康复可使心衰患者再住院率降低25%，冠心病患者死亡率降低20%以上。但目前我国心脏康复参与率不足10%，主要原因：

- 院内做康复周期短，出院后无人跟进
- 患者往返医院的依从性低（尤其术后恢复期行动不便）
- 基层社区缺少心脏康复专业能力

#### 3.2.2 项目流程

```
① 医院评估确诊（心超/心肺运动试验/冠脉CTA等）
      ↓ 医生开具居家康复处方
② 患者转诊至圆爱康居家康复体系
      ↓ 签约 → 领取设备
③ 居家康复执行（物理康复设备+远程指导）
      ↓ 每周1-2次康复训练，数据回传
④ 定期复评（回医院或社区）
      ↓
⑤ 康复周期结束 → 效果评估
```

#### 3.2.3 所需设备清单

| 环节 | 设备名称 | 参考价格 | 配置方式 |
|:---|:---|:---:|:---|
| **评估** | 心脏彩色超声 | 30-100万 | **建议医院自行配备**（常规设备，多数医院已有） |
| **评估** | 心肺运动试验（CPET） | 30-80万 | **建议有条件的医院采购**（心脏康复金标准） |
| **评估** | 运动负荷心电图 | 10-30万 | **建议社康/二级医院配备** |
| **院内治疗** | 心脏康复监护系统 | 20-50万 | **建议医院采购** |
| **居家康复** | 居家心脏康复训练设备 | — | **含在服务费内**，患者租赁使用，循环利用 |

> ⚠️ 心脏评估环节必须在医院完成，本项目以"医院开处方→第三方承接院外康复"模式运行

#### 3.2.4 服务内容

- 医院转诊对接机制
- 居家康复设备配置与指导
- 远程康复训练方案（运动处方）
- 康复数据监测与预警
- 定期复评安排

#### 3.2.5 结算模式

按康复周期（3个月/6个月）打包结算，或按月付服务费。

### 3.3 心理健康服务体系

#### 3.3.1 现状与痛点

认知功能障碍（含阿尔兹海默症早期、轻度认知障碍MCI）和ADHD是近年来基层卫生服务的新挑战。早期发现、早期干预可显著延缓病程进展，但：

- 认知筛查未纳入常规体检
- 缺乏高效的社区筛查工具
- 患者干预手段有限，院外管理几乎空白

#### 3.3.2 项目流程

```
① 社区认知筛查日（平板评估：MMSE/MoCA等标准化评估）
      ↓
② 阳性分层
      ↓
   ┌───────────┴───────────┐
   ▼                       ▼
轻度（居家数字疗法干预）    中重度（转院治疗）
   │                        │
   ▼                        ▼
数字疗法游戏训练         院内治疗（经颅电刺激等）
   │                        │
   ▼                        ▼
远程指导+评估            康复期可同步进行居家数字训练
```

#### 3.3.3 所需设备清单

| 环节 | 设备名称 | 参考价格 | 配置方式 |
|:---|:---|:---:|:---|
| **评估** | 认知评估系统（软件+平板） | 3-8万 | 可由第三方服务机构提供，或医院自行采购 |
| **院内治疗** | 经颅电刺激仪 | 10-30万 | **建议医院采购**（院内专业操作） |
| **居家康复** | 数字疗法训练软件 | — | **无需采购硬件**，患者自有手机/平板即可使用，含在服务费内 |

> ✅ 居家数字疗法无需额外硬件投入，患者使用自有设备即可开始训练

#### 3.3.4 服务内容

- 社区认知筛查日执行
- 阳性患者分层管理与转诊
- 居家数字疗法训练方案
- 远程训练跟进与效果评估
- 院内治疗后的延续性居家康复

#### 3.3.5 结算模式

按疗程结算（如12周/疗程），或按人头包干。

## 四、卫健局的采购方案建议

根据各区实际情况和预算，提供三种方案选择：

### A方案：轻量启动型（推荐首批试点项目采用）

**适用场景**：预算有限、希望先试点的区/街道

| 项目 | 说明 |
|:---|:---|
| **设备采购** | 零——不采购设备 |
| **第三方服务** | 圆爱康自带评估设备+人员进社区做筛查，居家康复设备含在服务费内 |
| **卫健局支出** | 仅按筛查例数+康复例数支付服务费 |
| **优势** | 零启动成本，快速验证效果，无设备闲置风险 |

**预估年度投入**：根据筛查规模和康复人数浮动，每人次康复服务费约XXX元（可根据实际协商）

### B方案：稳步推进型

**适用场景**：预算中等、有医院/社康支撑的区

| 项目 | 说明 |
|:---|:---|
| **设备采购** | 采购医院端核心评估设备（心超/CPET/经颅电刺激仪等） |
| **第三方服务** | 居家康复设备+服务由圆爱康提供 |
| **卫健局支出** | 设备采购费（一次性）+ 康复服务费（持续） |
| **优势** | 院内能力有提升，院外管理不操心 |

**预估设备投入**：约50-200万元（根据实际配置清单）

### C方案：全面覆盖型

**适用场景**：预算充足、希望建立完整体系的区

| 项目 | 说明 |
|:---|:---|
| **设备采购** | 采购全部院内评估+治疗设备 |
| **第三方服务** | 居家康复一体化全承包 |
| **卫健局支出** | 设备采购费 + 服务费 |
| **优势** | 院内外全流程自控，数据完整 |

**预估设备投入**：约200-500万元（根据实际配置清单）

## 五、预期成效

### 5.1 量化指标

| 指标 | 预期提升 |
|:---|:---:|
| 盆底疾病筛查覆盖率 | 提升至60%以上（试点社区） |
| 心脏康复参与率 | 提升至30%以上（目前不足10%） |
| 认知障碍早期发现率 | 提升至50%以上 |
| 慢病管理达标率 | 提升20-30个百分点 |
| 居家康复依从率 | 达到80%以上 |

### 5.2 社会效益

- **减少复发再住院**：科学居家康复可降低心脏术后再住院率20-25%
- **延缓疾病进展**：认知障碍早期干预可延缓病程1-3年
- **提升居民满意度**：在家就能康复，减少奔波
- **减轻医保支出**：康复管理做在前，住院费花在后

## 六、实施路径规划

```
第一阶段（第1个月）：试点启动
  ├── 选定1-2个试点社区
  ├── 现场调研、设备调配
  └── 社区宣传预热

第二阶段（第2个月）：筛查日启动
  ├── 社区健康角开始运营
  ├── 三大方向筛查日排期
  └── 阳性患者纳入管理

第三阶段（第3-6个月）：全面运营
  ├── 筛查持续开展
  ├── 居家康复患者进入康复周期
  └── 数据收集与效果评估

第四阶段（第7个月起）：复制推广
  ├── 总结试点经验
  ├── 推广至更多社区
  └── 形成区域性慢病管理网络
```
