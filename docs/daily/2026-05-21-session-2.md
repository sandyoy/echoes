## 话题一：电脑微信找不到 Hermes 的问题

**背景：** 用户手机微信置顶了 Hermes，但同一微信号的电脑微信上找不到 Hermes 的聊天窗口，搜索也搜不到。

**讨论过程：**
- 用户尝试在电脑微信搜索栏搜 Hermes/小龙虾，搜不到
- 用户尝试在手机发消息后查看电脑微信同步情况，仍然找不到
- 根源：Hermes 通过企业微信（WeCom）通道接入手机微信，电脑微信不支持显示企业微信外部联系人的对话框
- 这不是置顶或搜索能解决的问题，是平台接入方式的问题

**Hermes 提出的方案：**
- A: 接入电脑微信的通道（需要研究可行性）
- B: 换平台——同时接 Telegram 或 Signal，电脑上打开就能聊
- C: 用 Web 网页模式，浏览器访问

**用户回复："有什么需要我帮助的跟我说"** — 表现信任，让 Hermes 自己去解决

**决策：** Hermes 自行研究解决方案，用户不干预。
