# 「院内诊疗 + 居家康复」延续服务包合作方案

> 草案 · 供院长审阅
> 广州圆爱康生物科技有限公司

---

## 一、核心模式（一句话）

> **医院出方案 + 圆爱康出设备+远程执行 + 患者在家完成康复 = 三方共赢**

---

## 二、背景与痛点

### 医院面临的问题

- 患者出院后康复断档，术后 3-6 个月是关键期，但患者难以持续往返医院
- 科室康复床位有限，无法覆盖所有需要居家康复的患者
- 患者流失到院外找不规范的康复服务，效果不可控

### 圆爱康的定位

以**物联网康复设备 + 远程六师共管（康复师 / 护士 / 营养师 / 运动指导师 / 心理指导师 / 健康管理师）**为核心能力，成为医院康复科的"居家服务延伸单元"。

**不派人上门**——全部通过设备投放 + 远程指导完成。

---

## 三、服务流程

### 服务总览图

```
患者在院治疗/手术期间
       │
       ▼
【第一步】医生评估 → 推荐居家康复服务包
  主治医生评估患者情况：是否适合居家康复
  如果适合，向患者推荐圆爱康「居家延续康复服务包」
  （医生不额外收费——推荐本身就是诊疗的一部分，
   包含在患者正常的住院/就诊费用中）
       │
       ▼
【第二步】患者签约 → 付给圆爱康
  患者自愿签署圆爱康服务包，支付 2,980 元/期
  钱付给圆爱康，圆爱康开票（自费，不走医保）
  服务包含：居家康复方案 + 设备 + 远程六师共管
       │
       ▼
【第三步】居家执行（全部远程）
  圆爱康寄送康复设备到患者家中
  圆爱康六师团队远程一对一指导（视频/消息/电话）
  物联网康复数据实时回传，同步医院主管医生
  居家康复方案——由圆爱康按医生指导方向落地执行
       │
       ▼
【第四步】阶段性复评
  按周期（每月或关键节点）：
  患者回医院复诊 → 医生评估进展 → 调整方案
  复诊按医院正常流程收费（医保/自费）
```

### 关于收费，一句话说清楚

> **居家康复的"方案 + 设备 + 指导"全部打包在圆爱康服务包里，患者只向圆爱康付这一笔钱（2,980元）。住院/就诊的费用照常付给医院，复诊费照常付给医院——除此之外，没有第三笔、没有重复收费。**

### 远程执行的具体内容

| 服务模块 | 执行方式 | 频次 |
|---------|---------|------|
| 设备使用指导 | 首次视频教学 + 后续群内答疑 | 首次 1 次，后续按需 |
| 康复训练督导 | 远程视频指导 + 打卡监督 | 每日/隔日（按方案） |
| 康复数据监测 | 设备自动上传 + 异常预警 | 实时 |
| 营养/运动/心理指导 | 六师共管远程咨询 | 每周 1-2 次 |
| 医生远程知会 | 周期汇总报告同步主管医生 | 每周/每两周 |

## 四、钱流模型（核心）

> **关键理解**：患者只认识"圆爱康服务包"这一个院外价格。医院的部分是圆爱康的内部商业成本，与患者的交费无关。

### 钱怎么流

```
患者
  │
  │ 支付 2,980 元给圆爱康（服务包费，自费）
  │
  ▼
┌──────────────┐
│   圆爱康      │ ← 圆爱康向患者开票：居家康复服务包
│  (收费方)     │
└──────┬───────┘
       │
       │（以下为圆爱康内部成本分配，患者不感知）
       │
       ├──→ 医院协作费（50%-60%）← 圆爱康月结支付给医院
       ├──→ 六师远程服务（圆爱康承担）
       └──→ 设备成本 + 运营（圆爱康承担）
```

**简单说：**
- 患者看到的：**"圆爱康居家康复服务包 = 2,980 元"**，没有第二笔院外费用
- 圆爱康视角：收了 2,980 元，**50%-60%作为协作费支付给医院**，剩余覆盖六师远程服务、设备折旧、运营及利润
- 医院视角：**不直接向患者收取居家服务包费用**，圆爱康按完成例数月结协作费
- 补充：院内诊疗费、复诊费仍按医院现有流程收取，与圆爱康服务包无关

### 对患者的呈现（一句话）

> "您签约一个居家康复服务包，2,980 元。我们寄设备到您家，远程指导您完成康复。中间如果需要回医院复诊，按医院正常流程走就行。"

**不向患者展示"医院分了多少、设备分了多少"。**

### 对医院的结算

- 圆爱康向医院支付**技术协作费**，按月结算
- 支付比例：服务包费用的**50%-60%**（按具体病种和合作深度协商）
- 票据：医院向圆爱康开具"技术服务费"发票
- 结算依据：当月完成服务的患者例数

### 医院的项目怎么立

**推荐方式：不走物价审批，直接入"合作收入"科目**

- 医院不需要新立收费项目，不需要向物价局报批
- 医院收的是**圆爱康支付的技术协作费**，走财务科目：**"其他收入 — 合作服务收入"**
- 院内原有的康复项目、门诊价格不受任何影响

> **为什么能这么走？** 因为钱是圆爱康付给医院的，不是患者付给医院的。医院没有对患者多收一分钱、没有改任何一个医疗收费项目。

### 对院长来说，两个核心顾虑的回应

| 院长顾虑 | 回应 |
|---------|------|
| "圆爱康收了钱不付给医院怎么办？" | 结算条款写入协议：月度对账、次月 15 日前结清。逾期医院有权暂停合作——和其他供应商合作一样的商业保障 |
| "改收费项目会不会被审计？" | 不改任何医疗收费项目。医院收的是"技术协作费"，走其他收入科目。合法合规 |

## 六、合同框架（核心条款建议）

### 协议名称

**《居家康复延续服务包合作服务协议》**

### 关键条款表述

> **第三条 合作内容**
> 甲方（医院）为乙方（圆爱康）提供居家康复患者的康复方案制定、阶段性复评及医生远程指导等技术支持。
> 乙方负责为患者提供居家康复设备、远程康复指导服务及服务包运营，独立向患者收取服务包费用。
>
> **第五条 结算方式**
> 乙方按服务完成例数向甲方支付技术协作费，标准为人民币____元/例（或按服务包收入的____%结算）。
> 结算周期为每月一次，乙方于次月 15 日前支付上月全部结算金额。
>
> **第七条 责任划分**
> 乙方提供的居家康复服务属于院外延伸健康管理服务，服务过程中出现的与设备使用及远程指导相关的问题由乙方负责。
> 甲方负责患者院内诊疗及康复方案制定的医疗行为，按现有医疗规范承担相应责任。

### 重点提示

- 不出现"购买医疗服务"、"转卖服务"、"代收代付"等字眼
- 医院的角色定义为"技术指导方"——出方案、出评估
- 圆爱康的角色定义为"服务执行方"——出设备、出远程运营
- 结算写"协作费"或"技术服务费"，不写"分成"

## 七、对院长来说，三个免顾虑

| 院长顾虑 | 方案回应 |
|---------|---------|
| "会不会被审计查到违规收费？" | 医院不收居家服务包的费用，不涉及新增医疗收费项目。医院收到的是圆爱康支付的"技术协作费"，走其他收入科目，合法合规 |
| "圆爱康收了钱，医院的钱怎么保证？" | 结算写入协议条款，月度对账、次月 15 日前结清，具有法律效力。逾期医院可暂停合作 |
| "其他医院做过吗？" | 广东省内已有同类合作案例，国家政策明确支持。如需，圆爱康可提供其他合作医院的案例参考 |
