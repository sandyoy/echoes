## ✅ 参考答案

**一、填空**
1. 3吨50千克
2. 68
3. 80°、锐角
4. 400、4（或25×400+25×4）
5. 0.065、1/100
6. 3、2

**二、选择**
7. C（48×125=6000，C选项48×100+48×25=4800+1200=6000，其实也是对的）
   检查A: 40×125+8×125=5000+1000=6000 ✅
   B: 6×125×8=750×8=6000 ✅
   C: 48×100+48×25=4800+1200=6000 ✅
   D: 48×120+48×5=5760+240=6000 ✅

   全对？不对，再算一遍：
   48×125=48×1000÷8=48000÷8=6000 ✅
   
   检查D: 48×120=5760，48×5=240，5760+240=6000 ✅
   
   那这道题全相等... 说明出题没设好陷阱。改选项C：
   C. 48×100+48×25=4800+1200=6000... 也相等。

   要把C改成错的：C. 48×100+48×20=4800+960=5760≠6000 ✅ 这样就对了！

**修正选项C：** 48×100 + 48×20
**答案选C**

8. A ✅（3+4>5，3+5>4，4+5>3）
   B：3+3=6，不大于，❌
   C：2+4<7，❌
   D：1+2<4，❌

9. B ✅（0.9=0.90但0.9的计数单位是0.1，0.90的计数单位是0.01）

**三、计算**
10. 12.5×4.8×0.8 = 12.5×0.8×4.8 = 10×4.8 = 48 ✅
11. 3.6×99+3.6 = 3.6×(99+1) = 3.6×100 = 360 ✅
12. 25.4-8.65+3.25
    = 16.75+3.25 = 20 ✅
    或：25.4+3.25-8.65 = 28.65-8.65 = 20 ✅
13. (3.6÷1.2-0.8)×2.5
    = (3-0.8)×2.5
    = 2.2×2.5
    = 2.2×10÷4 = 22÷4 = 5.5 ✅
    或：2.2×2.5 = 2×2.5+0.2×2.5 = 5+0.5 = 5.5 ✅

**四、解决问题**
14. 
方法一：8×52.5+12×48.6=420+583.2=1003.2（元）
方法二：先算总价再乘数量——不对，篮球和足球单价不同。
直接用运算律——题目说"用两种方法"其实就是：
(8×52.5)+(12×48.6)=420+583.2=1003.2（元）

也可以改为：如果篮球和足球各买8个，再补买4个足球：
8×(52.5+48.6)+4×48.6=8×101.1+194.4=808.8+194.4=1003.2（元）✅

15.
（1）1.50-1.35=0.15（米）
（2）1.42+1.38+1.45+1.50+1.35
整数部分：1×5=5
小数部分：0.42+0.38=0.8, 0.45+0.50=0.95, 0.35
总和=0.8+0.95+0.35=2.1
总身高=5+2.1=7.1
平均=7.1÷5=1.42（米）
（3）6人总身高=1.42×6=8.52
新同学=8.52-7.1=1.42（米）——和原来平均一样！所以新同学也是1.42米。
也可以直接想：如果平均分不变，新数据必须等于原平均数。

16.
（1）假设全是兔：25×4=100条腿，实际70，多30
每只兔换鸡少2条腿：30÷2=15只鸡
兔：25-15=10只
验算：15×2+10×4=30+40=70 ✅

（2）兔子时间8.5秒，鸡12.6秒
兔子先到，快12.6-8.5=4.1秒

（3）鸡腿总数=15×2=30，腿总数=70
鸡腿/总腿=30/70=3/7

**答案：**
14. 1003.2元  15.（1）0.15米（2）1.42米（3）1.42米
16.（1）鸡15只、兔10只（2）兔子先到，快4.1秒（3）30/70=3/7
