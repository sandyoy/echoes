### 第1题（串联：平移 + 小数加减法）

下图中，一个点从A位置开始向右平移。
A(0) ——→ 第一次平移 3.5 格 ——→ 第二次平移 2.8 格 ——→ 第三次平移 ? 格 ——→ B(10.0)

（1）前两次平移后，点到了哪个位置？
（2）第三次需要平移多少格才能到达B点？
（3）列综合算式求第三次平移的距离。

**解题思路：**
（1）第一次：3.5格
第二次：3.5 + 2.8 = 6.3（格）

（2）10.0 - 6.3 = 3.7（格）

（3）综合算式：10.0 - (3.5 + 2.8) = 10.0 - 6.3 = 3.7（格）
这里用了**减法的性质**：a - (b + c) = a - b - c

**答案：** （1）6.3格 （2）3.7格 （3）10.0-(3.5+2.8)=3.7格

### 第2题（串联：平移 + 测量 + 小数计算）

一个长方形ABCD，A点坐标(0,0)，将长方形向右平移4.2厘米，再向上平移3.5厘米。

（1）平移后的A'点坐标是（___，___）
（2）如果原来长方形长5.3厘米，宽3.6厘米，平移后扫过的面积是多少？
（提示：平移是整体移动，计算面积可以用"新位置与原位置覆盖的矩形"来想，也可以用"平移过程中扫过的区域"来想）

**解题思路：**
（1）向右平移4.2厘米：x坐标+4.2
向上平移3.5厘米：y坐标+3.5
A'(0+4.2, 0+3.5) = (4.2, 3.5)

（2）平移扫过的区域像一个"L"形，不是简单矩形。但可以分解：
向右平移扫过：长5.3 × 宽4.2 = 22.26（平方厘米）
向上平移扫过：长(5.3+4.2) × 宽3.5 = 9.5 × 3.5 = 33.25（平方厘米）

这是对图形平移路径的理解。更标准的解法：
平移扫过的区域 = 原长方形面积 + 路径面积
原面积 = 5.3 × 3.6 = 19.08
但实际更复杂，小学阶段了解"图形平移不改变面积大小"即可。

**简化问题：** 
平移过程中，图形的**大小和形状不变**。原长方形面积=5.3×3.6=19.08（平方厘米）。
5.3×3.6 = 53×36÷100 = 1908÷100 = 19.08 ✅

**答案：** （1）(4.2, 3.5) （2）19.08平方厘米（平移不改变面积）

### 第3题（串联：小数计算 + 平均数 + 平移）

小明记录了家里一周（7天）的用电量（单位：千瓦时）：
周一 3.5  周二 4.2  周三 3.8  周四 4.0  周五 6.5  周六 7.2  周日 6.8

（1）这一周平均每天用电多少千瓦时？
（2）周五到周日的平均用电量比周一到周四的平均用电量多多少？
（3）把结果画成条形统计图（在纸上画）。如果周一的用电量向右"平移"增加1.5千瓦时，平均数会变成多少？

**解题思路：**
（1）总和 = 3.5+4.2+3.8+4.0+6.5+7.2+6.8
简便算法：整数部分(3+4+3+4+6+7+6)=33
小数部分(0.5+0.2+0.8+0+0.5+0.2+0.8)=3.0
总和=33+3=36（千瓦时）
平均=36÷7≈5.14（千瓦时）
（等一下：0.5+0.2+0.8+0+0.5+0.2+0.8=3.0是对的? 
0.5+0.2=0.7, +0.8=1.5, +0=1.5, +0.5=2.0, +0.2=2.2, +0.8=3.0 ✅）
平均=36÷7=5.142…保留一位小数≈5.1千瓦时

（2）周五~周日：(6.5+7.2+6.8)÷3 = 20.5÷3 ≈ 6.8（千瓦时）
周一到周四：(3.5+4.2+3.8+4.0)÷4 = 15.5÷4 = 3.875 ≈ 3.9（千瓦时）
多出：6.8 - 3.9 = 2.9（千瓦时）

（3）周一增加1.5后变成5.0
新总和=36+1.5=37.5
新平均=37.5÷7≈5.4（千瓦时）
比原来多：5.4-5.1=0.3（千瓦时）

**规律：** 增加的部分1.5被平均分到7天，平均数增加1.5÷7≈0.2
没错：5.36-5.14=0.22≈0.2 ✅

**答案：** （1）约5.1千瓦时 （2）多约2.9千瓦时 （3）约5.4千瓦时

### 第4题（串联：小数竖式 + 平移 + 观察物体）

小正方体棱长是0.5厘米。

（1）如果用4个这样的小正方体拼成一个长方体，这个长方体的长、宽、高各是多少？有几种拼法？
（2）每种拼法的体积相同吗？说明理由。
（3）0.5³ = 0.5 × 0.5 × 0.5 = ?

**解题思路：**
（1）4个棱长0.5cm的小正方体拼长方体：
- 拼法一：4排  → 长4×0.5=2cm, 宽0.5cm, 高0.5cm
- 拼法二：2排2层 → 长2×0.5=1cm, 宽0.5cm, 高2×0.5=1cm  
  或长1cm, 宽1cm, 高0.5cm
- 拼法三：单层2×2 → 长1cm, 宽1cm, 高0.5cm

（2）体积相同。因为都是4个小正方体，每个体积=0.5³=0.125cm³
总体积=4×0.125=0.5cm³ ✅（不管怎么拼，体积不变）

（3）0.5 × 0.5 = 0.25
0.25 × 0.5 = 0.125

**答案：** （1）有多种拼法，见解题思路 （2）体积相同，都是4个小正方体组成
（3）0.125

### 第5题（串联：小数 + 平均数 + 四则运算——综合思维题）

四名同学的身高分别是1.45米、1.38米、1.42米、1.47米。

（1）他们的平均身高是多少米？
（2）如果新加入一名同学，平均身高变成了1.44米，新同学的身高是多少米？
（3）新同学的身高比原来最高的人高还是矮？相差多少米？
（4）计算：新同学身高 - 原平均身高 = 这个差 ÷ 5 = 新的平均身高比原来提高了多少？

**用这个规律来验证分子分母关系。**

**解题思路：**
（1）1.45+1.38+1.42+1.47
简便方法：整数部分=1×4=4，小数部分=0.45+0.38+0.42+0.47
0.45+0.47=0.92，0.38+0.42=0.80，总和=0.92+0.80=1.72
总和=4+1.72=5.72
平均=5.72÷4=1.43（米）

（2）5人总身高=1.44×5=7.20（米）
原来4人总=5.72（米）
新同学=7.20-5.72=1.48（米）

（3）原来最高1.47米，新同学1.48米
1.48 > 1.47，比最高的还高。高出1.48-1.47=0.01（米）

（4）新同学身高-原平均=1.48-1.43=0.05（米）
0.05÷5=0.01（米）
新平均=1.44，原平均=1.43，提高0.01（米）✅

**关键规律：** 新加一个人的数据超过了平均数，平均数就会提高。提高的量 = （新数据 - 原平均数）÷ 总人数。

**答案：** （1）1.43米 （2）1.48米 （3）高0.01米 （4）0.05米、0.01米 ✅
