### 第1题（串联：鸡兔同笼 + 平均数 + 小数）

学校组织植树活动，四（1）班共45人，男生每人种3棵，女生每人种2棵，一共种了115棵。

（1）男生和女生各有多少人？
（2）全班平均每人种多少棵树？（结果保留一位小数）
（3）如果每棵树苗8.5元，女生种的树一共花了多少钱？

**解题思路：**
（1）鸡兔同笼：
假设全是男生：45×3=135棵
实际115棵，差：135-115=20棵
男换女，每人少种1棵（3-2=1）
女生：20÷1=20人
男生：45-20=25人

验算：25×3+20×2=75+40=115 ✅

（2）115÷45≈2.6（棵）——保留一位小数

（3）女生种了20×2=40棵
40×8.5=340（元）
40×8.5=40×8+40×0.5=320+20=340（用了乘法分配律）✅

**答案：** （1）男生25人，女生20人 （2）约2.6棵 （3）340元

### 第2题（串联：阶梯定价 + 小数加减法 + 平均数）

某市自来水收费方式如下表：

| 用水量（吨） | 单价（元/吨） |
|:------------:|:------------:|
| 不超过15吨   | 2.5 |
| 超过15吨部分 | 3.8 |

小明家这个月用了23吨水。

（1）应缴水费多少元？
（2）如果小明家这个月的水费是67.5元，用了多少吨水？
（3）上半年（6个月）平均每月水费47.5元，总水费多少元？

**解题思路：**
（1）15吨以内：15×2.5=37.5（元）
超过部分：23-15=8（吨），8×3.8=30.4（元）
总水费：37.5+30.4=67.9（元）

8×3.8=8×(4-0.2)=32-1.6=30.4 ✅

（2）先用足15吨：15×2.5=37.5（元）
超出部分水费：67.5-37.5=30（元）
超出水量：30÷3.8≈7.89（吨）
总用水量：15+7.89=22.89≈22.9（吨）

（3）47.5×6=285（元）
或者 47.5×2×3=95×3=285（元）✅

**答案：** （1）67.9元 （2）约22.9吨 （3）285元

### 第3题（串联：图形运动 + 周长 + 小数计算）

一个四边形经过平移后，原始位置的数据如下：
A(1.5, 2.3) → 向右平移3.5 → B(____, ____)
B → 向上平移2.7 → C(____, ____)

（1）填出B和C的坐标。
（2）如果A、B、C刚好构成一个直角三角形，且∠A=90°，已知AB=3.5，AC=____？
（提示：列式计算）
（3）三角形ABC的周长是多少？（AB+AC+BC的值，BC需要用勾股定理，小学改为直接给出BC=4.42）

**解题思路：**
（1）B(1.5+3.5, 2.3) = (5.0, 2.3)
C(5.0, 2.3+2.7) = (5.0, 5.0)

（2）∠A=90°，三角形ABC中：
AB是水平方向，长度=3.5（向右平移的距离）
AC是垂直方向，长度=2.7（向上平移的距离）
但注意，A到C不是直接相连的边。

直角三角形ABC，∠A=90°，AB是直角边=3.5，AC的长度：
A(1.5, 2.3)，C(5.0, 5.0)
AC = √[(5.0-1.5)² + (5.0-2.3)²] = √[3.5² + 2.7²] = √[12.25 + 7.29] = √19.54

这对四年级太难了。直接简化：**AB=3.5，BC=4.42，且已知∠A=90°，求AC和周长。**

（2）∠A=90°，在直角三角形ABC中，AB和AC是直角边，BC是斜边。
AC² = BC² - AB² = 4.42² - 3.5² = 19.5364 - 12.25 = 7.2864
AC = √7.2864 ... 不好算。

**再次简化——直接给直角边，算周长：**

A(1.5, 2.3)向右平移3.5到B，再向上平移2.7到C。

（1）B(5.0, 2.3)，C(5.0, 5.0)  ✅

（2）三角形ABC，A(1.5, 2.3)，B(5.0, 2.3)，C(5.0, 5.0)
AB = 5.0 - 1.5 = 3.5（水平距离）
AC = 5.0 - 2.3 = 2.7（垂直距离）
BC = 通过勾股定理（小学不要求），直接给出BC=4.42
周长 = AB + AC + BC = 3.5 + 2.7 + 4.42 = 10.62

（3）3.5 + 2.7 = 6.2，6.2 + 4.42 = 10.62 ✅

**答案：** （1）B(5.0, 2.3) C(5.0, 5.0) （2）3.5、2.7 （3）10.62

### 第4题（串联：运算律 + 小数 + 三角形的秘密——综合思维题）

不计算结果，判断下面每组中的两个算式是否相等。相等的打"√"，不相等的打"×"：

A. 3.5 × 99 + 3.5  ______  3.5 × 100
B. 8.8 × 125  ______  8 × 125 + 0.8 × 125
C. 56.2 - (6.2 + 18.5)  ______  56.2 - 6.2 - 18.5
D. 7.5 × 4 + 2.5 × 4  ______  (7.5 - 2.5) × 4

**然后完成下面这个填空题：**

在△ABC中，∠A=45°，∠B+∠C=_____°，这是一个_____三角形（在"锐角/直角/钝角"中选一个）。
∠A正好是∠B的一半，∠B=_____°，∠C=_____°。

这道题串联了________、________和________三个知识点。

**解题思路：**
A. 3.5×99+3.5 = 3.5×(99+1) = 3.5×100 ✅（乘法分配律逆用）
B. 8.8×125 = (8+0.8)×125 = 8×125+0.8×125 ✅（乘法分配律）
C. a-(b+c)=a-b-c ✅（减法的性质）
D. 7.5×4+2.5×4 = (7.5+2.5)×4 = 10×4 = 40
(7.5-2.5)×4 = 5×4 = 20
左边≠右边 ❌

填空部分：
∠B+∠C = 180° - 45° = 135°
三角形内角和180°。
∠A=45°，∠A是∠B的一半，所以∠B=90°
∠C = 180° - 45° - 90° = 45°

三个角：45°、90°、45°。这是一个**直角**三角形（也是等腰直角三角形）。

这道题串联了：**运算律 + 小数计算 + 三角形内角和**。

**答案：** A✅ B✅ C✅ D❌；填空：135°、直角、90°、45°；串联：运算律、小数计算、三角形内角和

### 第5题（串联：鸡兔同笼 + 小数大小比较 + 四则运算——大串烧）

**大串烧挑战题：**

一场知识竞赛有20道题。答对得5分，答错扣3分，不答得0分。

（1）小红答了全部20题，得了76分，她答对了几题？
（2）小刚答了18题（2题没答），分数比小红低2.5分，他得了多少分？答对了几题？
（3）小刚、小红、小明的平均分正好是80分，小明得了多少分？
（4）80分和76分相比，80 ÷ 76 ≈ ______（保留两位小数），这说明小明的分数是小红的______倍。

**解题思路：**
（1）小红：全对=100分，实际76分，少24分
答错一题比答对少得5+3=8分
答错：24÷8=3题
答对：20-3=17题
验算：17×5-3×3=85-9=76 ✅

（2）小刚比小红低2.5分：76-2.5=73.5分
他答了18题，设答对x题，答错(18-x)题
5x - 3(18-x) = 73.5
5x - 54 + 3x = 73.5
8x = 127.5
x = 15.9375 —— 不是整数，不对。

**检查：** "低2.5分"是指总分还是平均分？
假设是总分低2.5分：76-2.5=73.5，但答对题数必须是整数，算出来不是整数——说明2.5这个数字不合适。

**修正：** 小刚比小红低**4分**
76-4=72分
5x - 3(18-x) = 72
5x - 54 + 3x = 72
8x = 126
x = 15.75 —— 还不是整数。

**再修正：** 小刚得了**76分**，和小红一样。
5x - 3(18-x) = 76
5x - 54 + 3x = 76
8x = 130
x = 16.25 —— 还是不对。

问题在于：答18题，得76分，答对题数必须整数。
设答对a题：
5a - 3(18-a) = 76
5a - 54 + 3a = 76
8a = 130
a = 16.25

不是整数。那正确答案应该是：
5a - 3(18-a) = 某数，且a为整数。

试a=16：5×16-3×2=80-6=74
试a=17：5×17-3×1=85-3=82

所以小刚要么74分，要么82分。那改题目：小刚比小红低2分。

**修正版：**
（2）小刚答了18题（2题没答），得了74分。他答对了几题？
5a - 3(18-a) = 74
5a - 54 + 3a = 74
8a = 128
a = 16 ✅
答对16题，答错2题。

验算：16×5-2×3=80-6=74 ✅

（3）(76 + 74 + 小明)÷3 = 80
76+74+小明 = 240
小明 = 240-150 = 90分

（4）80分… 等一下，这里90分不是80分。
**修正：** 三人平均分=76+74+90÷3=240÷3=80 ✅
(76+74+90)÷3=240÷3=80 ✅

80÷76≈1.05（保留两位小数），小明是小红的90÷76≈1.18倍。

这里写着有点乱，直接给经过验证的最终版本：

**最终版本（第5题）：**

（1）小红答了全部20题，得76分。答对几题？答错几题？
（2）小刚答了18题（2题没答），得74分。他答对几题？答错几题？
（3）小红、小刚和小明三人平均分是80分，小明得了多少分？
（4）小明的分数是小红的几倍？（结果保留两位小数）

**解题思路：**
（1）全对100分，少24分
每错1题少8分（少得5分+再扣3分）
错：24÷8=3题，对：17题 ✅

（2）5×16-3×2=80-6=74 ✅
答对16题，答错2题。

（3）小红76+小刚74=150
小明的分=240-150=90（分）

（4）90÷76≈1.18

**答案：** （1）答对17题，答错3题 （2）答对16题，答错2题
（3）90分 （4）约1.18倍
