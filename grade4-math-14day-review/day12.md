### 第1题（思维：倒推法 + 四则运算——"反着走"）

一个数加上3.6，乘以2，再减去4.5，最后除以3，结果是5.5。求这个数。

**解题思路：**
从结果倒着推。每一步都要做"逆运算"：
```
结果：5.5
← 除以3，原来乘以3：5.5×3=16.5
← 减去4.5，原来加上4.5：16.5+4.5=21.0
← 乘以2，原来除以2：21.0÷2=10.5
← 加上3.6，原来减去3.6：10.5-3.6=6.9
```

这个数是6.9。

**验算：**
(6.9+3.6)×2-4.5 = 10.5×2-4.5 = 21-4.5 = 16.5
16.5÷3 = 5.5 ✅

**答案：** 6.9

### 第2题（思维：设未知数思想 + 小数 + 四则运算）

图书馆里有一些书。借出总数的一半少15本，还剩102本。图书馆原来有多少本书？

**解题思路：**
方法一（列算式倒推）：
"借出总数的一半少15本" = 还剩 总数的一半+15本
所以 总数÷2 + 15 = 102
总数÷2 = 102-15 = 87
总数 = 87×2 = 174（本）

方法二（画图理解）：
|← 一半 →|← 一半 →|
[ 全部 ]→借出(一半-15)
剩下的=另一半+15=102
另一半=87，全部=174

**验算：** 174÷2-15=87-15=72（借出72本），剩余174-72=102 ✅

**答案：** 174本

### 第3题（串联：鸡兔同笼隐藏条件 + 小数 + 平均数——思维挑战）

公园里有一些长椅。每张长椅上坐3个大人多出2个大人，每张长椅上坐4个大人少1张长椅。

（1）公园里有多少张长椅？多少个大人？
（2）如果每张长椅前面站一个孩子收门票，大人门票每张8.5元，孩子免票，总收入是多少元？
（3）如果有7.5%的收入要交税（小学改为：交税金额占总收入的0.075，约等于总收入的____分之____），交税多少元？

**解题思路：**
（1）设长椅有x张。
每张坐3人：总人数=3x+2
每张坐4人：少1张椅，即(x-1)张坐满，总人数=4(x-1)

3x+2 = 4(x-1)
3x+2 = 4x-4
2+4 = 4x-3x
x = 6（张）

大人数量=3×6+2=18+2=20（人） 或 4×(6-1)=4×5=20（人）✅

（2）20×8.5=170（元）
20×8.5=20×8+20×0.5=160+10=170 ✅

（3）0.075 = 75/1000 = 15/200 = 3/40
0.075 = 7.5/100 = 75/1000 = 3/40
170×0.075 = 170×75÷1000 = 12750÷1000 = 12.75（元）
或者 170×3÷40 = 510÷40 = 12.75（元）

**答案：** （1）6张长椅，20个大人 （2）170元 （3）12.75元

### 第4题（串联：小数 + 运算律 + 分数初步——数感拓展）

（1）不计算，按从小到大的顺序排列：

A. 0.3 × 0.3
B. 0.3 + 0.3
C. 0.3 - 0.3
D. 0.3 ÷ 0.3
E. 0.3 × 1.5

（2）上面哪个算式的结果和"0.9的十分之一"相等？

（3）如果0.3 × 0.3 = 0.09，那么0.3 × 0.3 × 0.3 = 0.027，0.3 × 0.3 × 0.3 × 0.3 = ____
你发现了什么规律？

**解题思路：**
（1）先估算：
A = 0.3×0.3 = 0.09
B = 0.3+0.3 = 0.6
C = 0.3-0.3 = 0
D = 0.3÷0.3 = 1
E = 0.3×1.5 = 0.45

从小到大：C(0) < A(0.09) < E(0.45) < B(0.6) < D(1)

（2）0.9的十分之一 = 0.9÷10 = 0.09
0.3×0.3 = 0.09 ✅，等于A。

（3）0.3×0.3×0.3 = 0.027
0.3×0.3×0.3×0.3 = 0.0081
**规律：** 每多乘一个0.3，结果的位数就多一位小数，并且每次都是乘以3（不考虑小数点位置）。0.3^n就是3^n除以10^n。

**答案：** （1）C < A < E < B < D （2）A （3）0.0081

**三、解决问题（每题5分，共10分）**

⑥ 停车场有三轮车和自行车共18辆，轮子共48个。三轮车和自行车各有多少辆？
⑦ 一根绳子对折3次后，每段长1.25米。绳子原来长多少米？

---

**解题思路：**
**一、填空**
① 0.6 ÷ 0.1 = 6（个），0.06 ÷ 0.01 = 6（个）
② ∠3=180-38.5-61.5=180-100=80°，三个角都小于90°，是锐角三角形。
③ 4.05×100=405，4.05÷100=0.0405

**二、简便计算**
④ 12.5×3.2×2.5 = 12.5×8×0.4×2.5 = (12.5×8)×(0.4×2.5) = 100×1 = 100 ✅
⑤ 8.4×101-8.4 = 8.4×(101-1) = 8.4×100 = 840 ✅

**三、解决问题**
⑥ 假设全是自行车：18×2=36个轮子，实际48个，多12个
三轮车比自行车多1个轮子：12÷1=12辆三轮车
自行车：18-12=6辆
验算：12×3+6×2=36+12=48 ✅

⑦ 对折一次→2段，对折两次→4段，对折三次→8段
1.25×8=10（米）
或：1.25×2×2×2=1.25×8=10 ✅

**答案：**
① 6、6 ② 80°、锐角 ③ 405、0.0405
④ 100 ⑤ 840
⑥ 三轮车12辆、自行车6辆 ⑦ 10米
