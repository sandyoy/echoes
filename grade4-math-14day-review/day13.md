### 📌 考试技巧一：审题三遍再动笔

有的孩子一看题就动笔，结果经常做错。正确的做法是：

**第一遍**（粗读）：知道题目在讲什么
**第二遍**（细读）：圈出关键数字和条件
**第三遍**（检验）：问自己"我理解了题意吗？"

**练习1**（专门练审题——三个条件一个没用）

题目：一个三角形三条边分别是3.5cm、4.2cm和6.8cm，它的高是2.4cm，求这个三角形的周长。

**审题思考：** 条件有4个：3.5cm、4.2cm、6.8cm、2.4cm。问题问"周长"——只用到三条边。高2.4cm是**多余条件**（面积才需要用到高，求周长不需要）。

**答案：** 3.5+4.2+6.8=14.5cm ✅

### 📌 考试技巧二：检查比做题更重要

**练习2**（检查自己的计算）

小明算了一道题：12.5 × 8.8 = 108，请帮他检查对不对。

**检查方法1（估算）：** 12.5×8=100，12.5×8.8应该比100大一点，大概是110左右。108距离110很近，有可能对。

**检查方法2（反算）：** 108÷12.5≈8.64，不是8.8 ❌ 所以错了。

**检查方法3（换一种方法算）：** 
12.5×8.8 = 12.5×(8+0.8) = 12.5×8+12.5×0.8 = 100+10 = 110 ✅
所以正确答案是110。

**答案：** 小明算错了，正确是110。

### 📌 考试技巧三：不会的题先跳过

考试时遇到不会的题，**不要死磕**。先跳过做后面的，最后再回来想。有时候做后面的题会给你启发。

**练习3**（卡住时换个思路）

题目：一个等腰三角形的周长是18.6cm，腰长是底边的2倍，底边长多少？

**思路1（直接列式）：** 
设底边x，腰=2x
x+2x+2x=18.6
5x=18.6
x=3.72（cm）✅

**思路2（如果不会列方程）：**
画图——等腰三角形，两条腰一样长，每条是底边的2倍。
相当于把周长分成5份（底边1份，两条腰各2份）
每份=18.6÷5=3.72（cm）
底边=1份=3.72cm ✅

**答案：** 3.72cm

### 📌 考试技巧四：单位+小数点的"致命陷阱"

**练习4**（专门练单位转换和小数点）

a) 5.4米 = （ ）米（ ）厘米 = （ ）厘米
b) 3吨50千克 = （ ）吨
c) 2.5平方千米 = （ ）公顷
d) 3050克 = （ ）千克（ ）克

**解题思路：**
a) 5.4米 = 5米 + 0.4米 = 5米 + 40厘米 = 5米40厘米
  5.4米 = 5.4×100 = 540厘米

b) 3吨50千克 = 3 + 50÷1000 = 3 + 0.05 = 3.05吨

c) 1平方千米=100公顷，2.5平方千米=2.5×100=250公顷

d) 3050克 = 3000克÷1000 + 50克 = 3千克50克

**答案：** a) 5米40厘米、540厘米 b) 3.05吨 c) 250公顷 d) 3千克50克

### 📌 考试技巧五：验算方法大全

做完题后一定要验算。学会不同的验算方法：

**练习5**——对下面四道题各用一种方法验算：

① 计算题验算：32.5 + 18.7 = 51.2
**验算方法：** 减法验算——51.2 - 32.5 = 18.7 ✅ 或 51.2 - 18.7 = 32.5 ✅

② 简便计算验算：25 × 44 = 25 × (40+4) = 1000 + 100 = 1100
**验算方法：** 换一种简便方法——25×44=25×4×11=100×11=1100 ✅

③ 鸡兔同笼验算：笼里有鸡和兔共10只，共28条腿，鸡6只兔4只。
**验算方法：** 代入检验——6×2+4×4=12+16=28 ✅

④ 单位换算验算：0.5平方千米=50公顷
**验算方法：** 反过来想——1公顷=0.01平方千米，50公顷=0.5平方千米 ✅

### 📌 考试技巧六：画图+列表帮大忙

**练习6**（抽象题靠画图）

一根绳子，第一次用去一半少0.5米，第二次用去剩下的一半多0.5米，最后还剩3米。绳子原来多长？

**解题思路：**
画图法：
```
原来：|←————————— 这一段 —————————→|
第一次用去：|← 一半少0.5米 →|
            [  用掉的  ][← 剩下的 →]
                        [多0.5米]
                      |← 剩下的一半 →|← 第二次用掉 →|
                      [剩下的一半...][多0.5米] 最后剩3米
```

倒推法：
最后剩3米
第二次：用去剩下的一半多0.5米 → 剩下一半少0.5米=3米
所以剩下一半=3.5米，第二次之前有7米
第一次：用去一半少0.5米 → 剩下一半多0.5米=7米
所以一半=6.5米，原来=13米

**验算：** 第一次用去：13÷2-0.5=6.5-0.5=6米，剩7米
第二次用去：7÷2+0.5=3.5+0.5=4米，剩3米 ✅

**答案：** 13米

### 🎯 今日总结：考试"锦囊"

1. **审题三遍再动笔**——圈数字、标条件、找多余
2. **不会的题先跳过**——留到最后，做不出也不慌
3. **做完一定验算**——用"逆向法"（加变减、乘变除）
4. **单位陷阱最多**——全换成统一单位再算
5. **画图列表最管用**——抽象题画图、鸡兔同笼列表
6. **检查小数点**——有没有少点或多点一位？
