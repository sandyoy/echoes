### 第1题（串联：三角形分类 + 小数的性质 + 逻辑推理）

下面是一个三角形的三个角的度数，但有两个数字被墨水遮住了。

```
∠1 = 90.0°   ∠2 = 4□.5°   ∠3 = □5.5°
```

已知这是**直角三角形**，∠1已经给出。请推理出被遮住的数字，并计算∠2和∠3的度数。

**解题思路：**
∠1=90.0°=90°，说明这是一个直角三角形。
∠2 + ∠3 = 180° - 90° = 90°（三角形内角和180°）

设∠2十位为a，∠3十位为b。
∠2 = (40+a) + 0.5 = 40.5 + a（但此处a是十位数字，实际上是40+10×a...错了）

重新理解：4□.5，十位是4，个位是未知数x，所以∠2 = 40 + x + 0.5 = 40.5 + x
□5.5，十位是未知数y，所以∠3 = 10y + 5 + 0.5 = 10y + 5.5

∠2 + ∠3 = (40.5 + x) + (10y + 5.5) = 46 + x + 10y = 90
x + 10y = 44

x是0-9的个位数，y是0-9的十位数（且y≥4因为∠3≥45.5否则总和不够……不，y可以是0-9）
如果y=4，则x=4，∠2=44.5°，∠3=45.5° ✅
如果y=3，则x=14，不可能
如果y=5，则x=-6，不可能

所以：∠2=44.5°，∠3=45.5°

**答案：** ∠2=44.5°，∠3=45.5°。三角形为直角三角形。

### 第2题（串联：轴对称图形 + 小数加减法 + 三角形）

（1）画出下面这个等腰三角形底边上的高，它也是这个三角形的对称轴。（此题在纸上画）

（2）测得这个等腰三角形的两条边长分别是3.8厘米和7.5厘米，那么第三条边的长度是多少厘米？
（提示：注意等腰三角形的特征和三角形三边关系）

（3）求出这个三角形的周长（用小数加法竖式计算）。

**解题思路：**
（1）等腰三角形底边上的高就是对称轴，也是顶角的角平分线。

（2）等腰三角形有两条边相等。
如果腰=3.8厘米，底=7.5厘米：3.8+3.8=7.6 > 7.5 ✅，7.5-3.8=3.7 < 3.8 ✅（两边之和大于第三边，两边之差小于第三边）
如果腰=7.5厘米，底=3.8厘米：7.5+7.5=15 > 3.8 ✅，7.5-7.5=0 < 3.8 ✅（但这样底太小，腰太大，也是可以的）

两种都有可能？不对，题目说了"等腰三角形"但没有说哪两条边相等。一般等腰三角形只说"两条边"，那第三条边必须等于其中一条。
所以第三种情况：7.5+3.8=11.3 > 7.5 ✅，7.5-3.8=3.7 < 7.5 ✅，这也成立。

**但实际上等腰三角形中相等的两条边是腰。** 所以有两个可能：
情况A：腰=3.8，腰=3.8，底=7.5 → 3.8+3.8=7.6>7.5 ✅ 能围成

### 第2题（修正版）

这个等腰三角形的两条边长分别是**3.8厘米**和**7.5厘米**，已知其中一条是腰，一条是底边。

（1）如果3.8厘米是腰，第三条边（底边）是多少？能围成三角形吗？
（2）如果7.5厘米是腰，第三条边（底边）是多少？能围成三角形吗？
（3）分别求出两种情况下三角形的周长。

**解题思路：**
（1）腰=3.8，底=7.5。三边3.8、3.8、7.5。
3.8+3.8=7.6 > 7.5 ✅ 能围成。周长=3.8+3.8+7.5=15.1（厘米）

（2）腰=7.5，底=3.8。三边7.5、7.5、3.8。
7.5+3.8=11.3 > 7.5 ✅ 能围成。周长=7.5+7.5+3.8=18.8（厘米）

（3）两种情况都能围成三角形（都满足两边之和大于第三边）。

**答案：** （1）7.5厘米，能围成，周长15.1厘米 （2）3.8厘米，能围成，周长18.8厘米

### 第3题（串联：小数乘除法（结合运算律）+ 轴对称图形数量）

（1）下面图形各有多少条对称轴？
- 等边三角形：___条
- 正方形：___条
- 长方形：___条
- 圆：___条

（2）如果每个对称轴之间相隔的角度都相等，等边三角形的两条对称轴之间的最小夹角是多少度？
（3）利用上面结果算：0.25 × 4 - 0.5 × 2 = ？

**解题思路：**
（1）等边三角形3条、正方形4条、长方形2条、圆无数条。

（2）等边三角形有3条对称轴，把圆分成3等份。
360° ÷ 3 = 120°（两条对称轴的夹角）

（3）0.25 × 4 - 0.5 × 2 = 1 - 1 = 0 ✅

**答案：** （1）3条、4条、2条、无数条 （2）120° （3）0

### 第4题（串联：小数计算 + 三角形内角和 + 观察物体）

从上面观察一个物体，看到的是一个三角形。这个"小三棱柱"的两个底角分别是36.5°和58.5°。

（1）问第三个角是多少度？
（2）如果把这个三角形按角分类，它是什么三角形？
（3）三个角的平均数是多少度？

**解题思路：**
（1）180° - 36.5° - 58.5° = 180° - 95° = 85°
注意小数竖式对齐：
```
 180.0
- 36.5
-------
 143.5
- 58.5
-------
  85.0 → 85°
```

（2）三个角：36.5°、58.5°、85°。都小于90°，所以是**锐角三角形**。

（3）三个数的平均数 = 180° ÷ 3 = 60°（三角形内角和180°，平均每个角60°）
也可以用小数算：180.0 ÷ 3 = 60.0 ✅

**答案：** （1）85° （2）锐角三角形 （3）60°

### 第5题（串联：四则运算 + 三角形 + 小数—综合思维题）

一根铁丝刚好可以围成一个边长0.85米的等边三角形。

（1）这根铁丝有多长？
（2）如果用这根铁丝围成一个正方形，正方形的边长是多少米？
（3）正方形的面积是多少平方米？比围成的等边三角形面积大还是小？（这个需要想：等边三角形面积不知道，但可以比较"同样周长下正方形和三角形谁面积大"）
（4）假如正方形的边长用小数表示，它的面积能否用运算律简便计算？

**解题思路：**
（1）等边三角形三边相等，铁丝长 = 0.85×3 = 2.55（米）

（2）正方形周长 = 铁丝长 = 2.55米
正方形边长 = 2.55 ÷ 4 = 0.6375（米）

（3）正方形面积 = 0.6375 × 0.6375，不太好算。
换个思路：在周长相同的情况下，**同样周长，圆的面积>正方形面积>三角形面积**。
所以正方形面积 > 等边三角形面积。

如果要算：0.6375² 对四年级太难。可以简化思考：
0.6375 × 4 = 2.55，0.6375 = 2.55 ÷ 4
面积 = 2.55 ÷ 4 × 2.55 ÷ 4 = 2.55² ÷ 16
2.55² = (2.5+0.05)²... 太复杂。

**简化此题：** 改为边长0.25米的等边三角形，铁丝长=0.75米。
正方形边长=0.75÷4=0.1875米。面积=0.1875²，也不整。

**再简化：** 铁丝刚好围成一个边长**0.8米**的等边三角形。

（1）铁丝长 = 0.8 × 3 = 2.4（米）

（2）正方形边长 = 2.4 ÷ 4 = 0.6（米）

（3）正方形面积 = 0.6 × 0.6 = 0.36（平方米）
等边三角形面积不好算（小学不要求等边三角形面积公式）。
**关键思维：** 同样周长下，正方形的面积 > 三角形的面积。

（4）0.6 × 0.6 = 6 × 6 ÷ 100 = 36 ÷ 100 = 0.36 ✅
用了运算律中的巧算：把小数×小数转化为整数×整数再除以100。

**答案：** （1）2.4米 （2）0.6米 （3）0.36平方米，正方形比等边三角形面积大
（4）0.6×0.6=6×6÷100=36÷100=0.36
