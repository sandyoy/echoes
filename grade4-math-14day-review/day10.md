### 第1题（串联：轴对称 + 小数 + 三角形）

（1）等腰直角三角形的一个锐角是多少度？另一个锐角呢？
（2）等腰直角三角形有____条对称轴。
（3）如果等腰直角三角形的直角边长度是4.5厘米，它的周长是多少厘米？（提示：斜边≈6.36厘米，用小数加法计算）
（4）如果用9个完全一样的等腰直角三角形拼成一个正方形，这个正方形的边长是多少厘米？面积呢？

**解题思路：**
（1）等腰直角三角形：一个角90°，两个锐角相等
180°-90°=90°，90°÷2=45°
两个锐角各45°。

（2）等腰直角三角形有**1条**对称轴（从直角顶点向斜边作垂线）。

（3）周长=4.5+4.5+6.36=15.36（厘米）
竖式：
```
 4.5
 4.5
+6.36
-----
15.36
```
注意：小数点对齐，4.5=4.50

（4）拼成正方形怎么拼？两个等腰直角三角形可以拼成一个正方形（斜边对斜边）。
每个等腰直角三角形斜边=6.36厘米。
9个三角形... 可以4个三角形拼成1个正方形（暂不考虑怎么拼）。

实际上与题目无关的复杂了，简化：

用**两个完全一样**的等腰直角三角形拼成一个正方形：
斜边是正方形的对角线，或者两条直角边拼成正方形的边？

两个等腰直角三角形拼正方形的方法：把它们的斜边重合，拼成一个正方形？不对，这样拼出来是个菱形。
**正确拼法：** 把等腰直角三角形的直角顶点和直角顶点重合…不对。
**标准拼法：** 两个完全一样的等腰直角三角形，把斜边拼在一起——但斜边长度就是正方形的边长的√2倍，不是简单拼法。

**最直接的拼法：** 两个等腰直角三角形，直角边挨着直角边（直角顶点在一起），拼成一个大的正方形——不对，那样拼出来是长方形。

**真实标准拼法：** 取等腰直角三角形，把另外一个相同的三角形沿着它的高翻转，让它们的直角边对齐——其实最简单：**两个等腰直角三角形可以拼成一个正方形当且仅当它们是等腰直角三角形**——直角边=4.5，斜边≈6.36，直角边拼直角边得到的是一个四边形，不是正方形。

实际上：两个等腰直角三角形沿斜边拼在一起，得到的四边形是**正方形**，因为四个角都是直角，四条边都等于直角边长4.5cm。

好了：一个正方形由2个等腰直角三角形拼成。
边长=4.5cm，面积=4.5×4.5=20.25（cm²）
9个三角形 → 4.5个正方形... 不整数。

### 第1题（修正简化版）

等腰直角三角形的直角边长度是3厘米。

（1）两个锐角各是多少度？
（2）自己画一条对称轴。
（3）周长是多少？（斜边按4.2厘米算）
（4）两个完全一样的等腰直角三角形可以拼成一个什么图形？

**解题思路：**
（1）(180-90)÷2=45°，两个锐角各45°

（2）从直角顶点向斜边画垂线，就是对称轴。

（3）3+3+4.2=10.2（厘米）

（4）两个一样的等腰直角三角形可以拼成**正方形**（斜边对斜边）或者**更大的等腰直角三角形**（直角边对直角边）。

**答案：** （1）各45° （3）10.2厘米 （4）正方形或等腰直角三角形

### 第2题（串联：平移 + 小数 + 运算律——图形平移与计算）

一个长方形在平面上的四个顶点坐标为：
A(1.2, 1.5)、B(5.2, 1.5)、C(5.2, 3.5)、D(1.2, 3.5)

（1）这个长方形的长和宽各是多少？
（2）面积是多少平方米？用乘法结合律或分配律简便计算。
（3）将这个长方形先向右平移2.8，再向上平移1.5，新长方形A'B'C'D'的顶点坐标是多少？
（4）平移前后，长方形的面积变了吗？说明理由。

**解题思路：**
（1）长=5.2-1.2=4.0，宽=3.5-1.5=2.0
长4.0米，宽2.0米。

（2）面积=4.0×2.0=8.0（平方米）
简便：4×2=8，再根据小数位数→8.0 ✅

（3）向右2.8：所有x坐标+2.8
向上1.5：所有y坐标+1.5

A'(1.2+2.8, 1.5+1.5) = (4.0, 3.0)
B'(5.2+2.8, 1.5+1.5) = (8.0, 3.0)
C'(5.2+2.8, 3.5+1.5) = (8.0, 5.0)
D'(1.2+2.8, 3.5+1.5) = (4.0, 5.0)

（4）面积不变。平移是一种刚性运动，图形的大小和形状都不变。
原面积=8.0，新面积=(8.0-4.0)×(5.0-3.0)=4.0×2.0=8.0 ✅

**答案：** （1）长4.0米，宽2.0米 （2）8.0平方米 （3）见解题思路 （4）不变

### 第3题（串联：平均数进阶 + 小数 + 逻辑推理——"隐藏数据"题）

① 小刚三科平均分是87.5分。

（1）小刚的英语成绩是多少？
（2）小丽的数学成绩是多少？
（3）小丽的数学、语文、英语平均分是多少？（结果保留一位小数）

365÷4=91.25≠90.75

所以小丽数学=89.0？但95.5-89=6.5≠4.5

### 第4题（串联：鸡兔同笼 + 小数乘除 + 四则运算——反向思维）

松鼠妈妈采松子，晴天每天采6.5千克，雨天每天采4.5千克。一共采了8天，共采了44千克。

（1）这8天中，晴天和雨天各有多少天？
（2）晴天平均每天比雨天多采多少千克？
（3）如果采的总量不变，但8天中有5天是雨天，需要增加多少千克的总量？（或减少多少天？）

**解题思路：**
（1）鸡兔同笼：
假设全是晴天：8×6.5=52（千克）
实际44千克，差52-44=8（千克）
每天晴比雨多：6.5-4.5=2（千克）
雨天：8÷2=4（天）
晴天：8-4=4（天）

验算：4×6.5+4×4.5=26+18=44 ✅

（2）6.5-4.5=2（千克）

（3）5天雨天，3天晴天：
总量=5×4.5+3×6.5=22.5+19.5=42（千克）
目标44千克，还差44-42=2千克。
需要增加总量2千克，或者把1天雨天改成晴天——多采2千克。

**答案：** （1）晴天4天、雨天4天 （2）2千克
（3）还差2千克，可以把1天雨天改为晴天

### 第5题（串联：四则运算 + 运算律 + 三角形 + 小数——大总结）

**考考你能不能一眼看出规律：**

1. 计算：
   A = 1.25 × 8.8
   B = 1.25 × 88 ÷ 10
   C = 1.25 × 8 + 1.25 × 0.8
   D = (1.25 × 8) × 1.1

2. A、B、C、D中，哪些相等？哪些不相等？

3. 如果把1.25改成0.25，8.8改成4.8，上面的规律还成立吗？自己验算一下。

4. 从上面你能总结出几条"算式的等价变形式"吗？写下来。

**解题思路：**
1. 计算：
   A = 1.25×8.8 = 11 × 1.25×0.8... 实际：1.25×8.8=11
   方法：1.25×8+1.25×0.8=10+1=11 ✅
   
   B = 1.25×88÷10 = (1.25×88)÷10 = 110÷10=11 ✅
   
   C = 1.25×8+1.25×0.8 = 10+1=11 ✅
   
   D = (1.25×8)×1.1 = 10×1.1=11 ✅

2. 全部相等！都是11。

3. 验证0.25×4.8：
   0.25×4.8 = 0.25×(4+0.8) = 1+0.2 = 1.2
   0.25×48÷10 = 12÷10 = 1.2 ✅
   0.25×4+0.25×0.8 = 1+0.2 = 1.2 ✅
   (0.25×4)×1.2 = 1×1.2 = 1.2 ✅
   规律也成立！

4. **等价变形式（四种思路解同一道题）：**
   - **拆分法：** a×(b+c) = a×b + a×c（乘法分配律）
   - **扩缩法：** a×b = (a×c)×(b÷c)（把一个数扩大，另一个缩小相同倍数）
   - **结合法：** 把b拆成c×d，a×b = a×c×d（乘法结合律）
   - **分解法：** a×b×c 与 a×(b×c) 等价

**答案：** 四者全部相等，都是11。规律在四年级期末考中非常实用——"一题四解，殊途同归"。
