### 第1题（易错点：运算律逆用——经典的"看似对其实错"）

**判断对错**，并说明理由。如果错了，在旁边写出正确答案。

（1）125 × 88 = 125 × 80 + 125 × 8  （  ）
（2）125 × 88 = 125 × 8 × 11  （  ）
（3）56 × 99 = 56 × 100 - 1  （  ）
（4）56 × 99 = 56 × 100 - 56  （  ）
（5）8.4 ÷ 0.4 ÷ 0.5 = 8.4 ÷ (0.4 × 0.5)  （  ）
（6）8.4 ÷ 0.4 ÷ 0.5 = 8.4 ÷ 0.4 × 0.5  （  ）

**解题思路：**
（1）✅ 正确。乘法分配律：125×(80+8)=125×80+125×8

（2）✅ 正确。先拆数再结合：125×8×11=1000×11=11000

（3）❌ 错误！56×99 = 56×(100-1) = 56×100 - 56×1 = 5600-56 = 5544
   而56×100-1=5600-1=5599，差55

（4）✅ 正确。就是(3)的正确写法

（5）✅ 正确。除法的性质：a÷b÷c = a÷(b×c)

（6）❌ 错误！除法的性质是变号，不能把÷变成×
   8.4÷0.4÷0.5 = 8.4÷(0.4×0.5) = 8.4÷0.2 = 42
   而8.4÷0.4×0.5 = 21×0.5 = 10.5
   结果完全不一样。

**答案：** ⚠️ 记住：
- 分配律：a×(b+c)=a×b+a×c，但a×(b+c)≠a×b+c（少了×c）
- 除法性质：a÷b÷c=a÷(b×c)✅  a÷b÷c≠a÷b×c❌

### 第2题（易错点：小数的"0"——末尾0 vs 中间0）

（1）下面各数中，哪些"0"可以去掉？哪些不能？
   3.50  0.508  5.080  10.05  0.800  500.5

（2）将上面可以化简的数化简。
（3）在0.508中，如果去掉小数点后的第一个"0"，变成了____，这个数变____了（大/小）。
（4）3.50和3.5比较，大小____，但计数单位____（相同/不同）。
   3.50的计数单位是____，3.5的计数单位是____。

**解题思路：**
（1）和（2）：
- 3.50 → 末尾0可去 → 3.5 ✅
- 0.508 → 没有末尾0，不能去 ❌（中间的0决定了数位）
- 5.080 → 末尾0可去 → 5.08 ✅
- 10.05 → 中间0不能去 ❌
- 0.800 → 末尾可去 → 0.8 ✅
- 500.5 → 整数部分中间0不能去 ❌

（3）0.508去掉第一个0→0.58。0.508是508个千分之一，0.58是580个千分之一（或58个百分之一），变大了。

（4）大小相同（3.50=3.5），但计数单位不同。
3.50的计数单位是0.01（百分之一），有350个这样的单位。
3.5的计数单位是0.1（十分之一），有35个这样的单位。

**答案：** 见解题思路。**关键记忆：小数末尾添0或去0，大小不变；中间0不能去！**

### 第3题（易错点：三角形三边关系的"陷阱"——两边之和大于第三边要验全部三组）

判断下面每组线段能否围成三角形。能的打"√"，不能的说明理由。

（1）3cm、4cm、7cm  （  ）
（2）4cm、5cm、9cm  （  ）
（3）0.3cm、0.4cm、0.8cm  （  ）
（4）2.5cm、3.8cm、6.2cm  （  ）
（5）4.2cm、4.2cm、4.2cm  （  ）
（6）5cm、5cm、11cm  （  ）

**解题思路：**
⚠️ **三边关系**：任意两边之和 > 第三边。要验三组！

（1）3+4=7 = 7，不大于 → ❌（等于也不行！）

（2）4+5=9 = 9，不大于 → ❌

（3）0.3+0.4=0.7 < 0.8 → ❌

（4）2.5+3.8=6.3 > 6.2 ✅
   2.5+6.2=8.7 > 3.8 ✅
   3.8+6.2=10.0 > 2.5 ✅ → ✅能围成

（5）4.2+4.2=8.4 > 4.2 ✅ 三组都成立 → ✅ 等边三角形

（6）5+5=10 < 11 → ❌（两腰之和小于底边，围不成）

**答案：** （1）❌ （2）❌ （3）❌ （4）✅ （5）✅ （6）❌
**记忆口诀：** "两边之和大于第三边，三组都要验一遍。"

### 第4题（串联：鸡兔同笼变式 + 小数 + 平均数——易错综合）

停车场里，汽车（4轮）和摩托车（2轮）共30辆，轮子总数是96个。

（1）汽车和摩托车各有多少辆？
（2）如果每辆汽车长4.5米，每辆摩托车长2.0米，所有车辆排成一队，车队有多长？
（3）如果把汽车和摩托车的平均长度作为"标准车长"，求标准车长是多少米？

**常见错误预警：**
- 第（1）题容易忽略：车轮数量不仅是多和少的关系
- 第（3）题容易误算为"两种车长加起来除以2"，这种算法错在哪里？

**解题思路：**
（1）鸡兔同笼：
假设全是摩托车：30×2=60个轮子
实际96个，多出96-60=36个
汽车比摩托多：4-2=2个轮子
汽车：36÷2=18辆
摩托：30-18=12辆

验算：18×4+12×2=72+24=96 ✅

（2）车队长度 = 18×4.5 + 12×2.0 = 81 + 24 = 105（米）

（3）**正确算法：** 总长度÷总车辆数 = 105÷30 = 3.5（米）
**❌ 常见的错误算法：**(4.5+2.0)÷2 = 6.5÷2 = 3.25（米）
**为什么错？** 因为汽车和摩托车的**数量不同**！平均长度要考虑重量——汽车多，平均长度就更接近汽车。这就是**加权平均**的概念。

验证：直接用分配律展开——
(18×4.5+12×2.0)÷30 = [18×4.5+12×2.0]÷30 = (81+24)÷30 = 105÷30 = 3.5

如果用(4.5+2.0)÷2，相当于假设汽车和摩托车一样多，但实际不一样。

**答案：** （1）汽车18辆，摩托12辆 （2）105米 （3）3.5米（加权平均不能简单÷2）

### 第5题（易错点+串联：小数加减竖式对齐 + 运算律 + 逻辑推理——大坑题）

**认真看题，这道题有三个"坑"：**

小马虎在计算"3.56 + 1.8"时，把加号看成了减号。

（1）他算出的错误结果是多少？
（2）正确结果与错误结果相差多少？
（3）如果他把"1.8"看成了"18"来计算加法，正确结果与错误结果又相差多少？
（4）请用运算律解释为什么（2）和（3）的差值不同。

**解题思路：**
（1）3.56 - 1.8 = 3.56 - 1.80 = 1.76
⚠️ **第一个坑：** 小数减法竖式对齐——小数点对齐！1.8对齐到3.56的百分位要补0。
```
 3.56
-1.80
-----
 1.76
```

（2）正确=3.56+1.8=5.36
差值=5.36-1.76=3.60
或者直接用运算律想：正确比错误多了"两个1.8"（因为把-1.8变成+1.8，差了2×1.8）
2×1.8=3.6 ✅

（3）正确=3.56+1.8=5.36
错误=3.56+18=21.56
⚠️ **第二个坑：** 3.56+18，18没有小数点，对齐个位：
```
  3.56
+18.00
-------
 21.56
```
差值=21.56-5.36=16.20
或者：18-1.8=16.2 ✅

（4）**为什么差值不同？**
（2）中是减→加的变化：差的是2×1.8=3.6（因为符号反了）
（3）中是1.8→18的变化：差的是16.2（因为数字大小变了）
⚠️ **第三个坑：** 两者不是一个类型的错误。一个是"符号错"，一个是"数位错"。符号错是加变减，数位错是小数点位置错了。

所以小数计算最关键的是：**小数点对齐**和**数位要清**。

**答案：** （1）1.76 （2）3.6 （3）16.2 （4）两种错误本质不同
