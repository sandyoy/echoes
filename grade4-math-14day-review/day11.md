### 第1题（串联：观察物体 + 小数 + 平均数）

用棱长1厘米的小正方体搭了一个立体图形，从三个方向看到的形状如下：

从正面看： 从上面看：  从左面看：
  □□        □□□       □□
  □□         □□        □□

（1）搭这个立体图形最少需要几个小正方体？最多需要几个？
（2）如果这个小正方体是**0.5元**一个，最少需要的总价是多少？最多需要的总价是多少？
（3）最少和最多总价的平均数是多少元？

**解题思路：**
（1）从上面看是3列3行... 不对，上面写的是□□□和□□，那就是5个格子。
正面是两行两列（4个位置），上面是两行三列+一行两列... 

让我统一理解：
正面：□□ 两行两列（4个位置）
      □□
上面：□□□ 两行（第一行3个，第二行2个，共5个位置）
       □□
左面：□□ 两行两列（4个位置）
      □□

从上面看确定了俯视图有5个格子。最少需要5个小正方体（每个格子放1个）。
最多要看正面和左面是否一致——正面看到的2行2列和左面2行2列都与俯视图吻合。
所以最少5个，最多也是5个（每个格子只放1个，没有叠放的情况）。

但正面要求两行都看到，如果某个格子只放一层，正面还能看到第二层吗？
如果上面看有5个格子，正面看是2行2列，说明从正面看每列至少有2个...
正面的两行两列——意味着从正面看有2行（上下2层），2列（左右2列）。
上面看有3列（□□□ | □□)，所以正面应该看到3列，但描述是2列——矛盾了。

我这描述有问题。改一下描述让它们自洽：

从正面看：  从上面看：  从左面看：

（1）最少需要4个小正方体（上面看到3个位置，左面看到2列，正面看到3列，说明是3×2的矩形布局，最少就是3×2=6个... 也得统一一下）

算了，不改观察物体的几何描述了，直接换成简单版：

**简化：** 用棱长1厘米的小正方体搭了一个立体图形，用了8个小正方体搭成一个2×2×2的大正方体。

（1）这个大正方体的棱长是多少厘米？
（2）如果每个小正方体重0.125克，大正方体总重量是多少克？
（3）从正面、上面、左面看，各看到几个小正方形面？
（4）看到的三个面的总面积之和是多少平方厘米？

**解题思路：**
（1）2个小正方体排一排是2厘米，所以大正方体棱长=2厘米。

（2）8 × 0.125 = 1（克）（因为8×125=1000，所以8×0.125=1 ✅）

（3）从每个方向看，都是2×2=4个小正方形面。
正面：4个，上面：4个，左面：4个。

（4）每个小正方形面面积=1×1=1（平方厘米）
三个面一共：4+4+4=12（平方厘米）
总面积=12平方厘米。

**答案：** （1）2厘米 （2）1克 （3）各4个 （4）12平方厘米

### 第2题（串联：复式条形统计图 + 小数 + 平均数）

下面是四（1）班第一、二小组期中数学成绩统计表：

| 组别 | 90-100分 | 80-89分 | 70-79分 | 60-69分 |
|:----:|:--------:|:-------:|:-------:|:-------:|
| 一组 |    5     |    3    |    2    |    1    |
| 二组 |    4     |    5    |    1    |    0    |

（1）一组和二组各有多少人？
（2）如果90-100分按平均95分算，80-89分按平均84.5分算，70-79分按平均74.5分算，60-69分按平均64.5分算，求一组和二组的平均分各是多少？（保留一位小数）
（3）哪个组的平均分更高？高多少？

**解题思路：**
（1）一组：5+3+2+1=11人
二组：4+5+1+0=10人

（2）一组总分 = 5×95 + 3×84.5 + 2×74.5 + 1×64.5
= 475 + 253.5 + 149 + 64.5
= 475+253.5=728.5, 728.5+149=877.5, 877.5+64.5=942
一组平均 = 942÷11 ≈ 85.6（分）

二组总分 = 4×95 + 5×84.5 + 1×74.5 + 0
= 380 + 422.5 + 74.5 = 877
二组平均 = 877÷10 = 87.7（分）

（3）87.7 > 85.6，二组高。高出87.7-85.6=2.1（分）

**答案：** （1）一组11人，二组10人 （2）一组≈85.6分，二组=87.7分
（3）二组高，高2.1分

### 第3题（串联：运算律综合运用 + 小数的加减 + 数感——"凑整"思维）

不列竖式，用简便方法计算：

① 9.9 + 99.9 + 999.9 + 0.3
② 8.8 + 8.88 + 8.888 + 0.432
③ 1.25 + 2.25 + 3.25 + … + 8.25（等差数列求和）

**解题思路：**
① 9.9+0.1=10，99.9+0.1=100，999.9+0.1=1000
0.3 = 0.1+0.1+0.1
所以：9.9+99.9+999.9+0.3 = (9.9+0.1)+(99.9+0.1)+(999.9+0.1) = 10+100+1000 = 1110 ✅

② 8.8+0.2=9，8.88+0.12=9，8.888+0.112=9
0.432=0.2+0.12+0.112
所以：(8.8+0.2)+(8.88+0.12)+(8.888+0.112) = 9+9+9 = 27 ✅

③ 这是等差数列：1.25, 2.25, 3.25, ..., 8.25
首项1.25，末项8.25，共8项
和 = (首项+末项)×项数÷2 = (1.25+8.25)×8÷2 = 9.5×4 = 38 ✅
**或者**：先把整数部分相加：(1+2+...+8)=36
小数部分相加：0.25×8=2
总和=36+2=38 ✅

**答案：** ①1110 ②27 ③38

### 第4题（串联：鸡兔同笼 + 小数 + 平均数——难度升级）

共有20道口算题。做对一道得5分，做错一道扣2分，不做不得分也不扣分。

小明最后得了83分，已知他没做的题比做错的题多2道。

（1）小明做对、做错、没做的各多少道？
（2）如果他每道做对的题平均用时8.5秒，做错的题平均用时6.2秒，总用时多少秒？
（3）平均每题用时多少秒？（保留一位小数）

**解题思路：**
（1）设做错x道，则没做(x+2)道，做对20-x-(x+2)=18-2x道
得分 = 5×(18-2x) - 2×x = 90 - 10x - 2x = 90 - 12x = 83
12x = 7
x = 7/12 ≈ 0.583

不是整数！说明假设有问题。再想想——"没做的比做错的题多2道"这个条件导致x必须是整数。

调整：设做错a道，没做(a+2)道，做对20-a-(a+2)=18-2a道
5(18-2a) - 2a = 83
90-10a-2a=83
90-12a=83
12a=7
a=7/12 ❌ 不是整数

说明83分配这个条件无整数解。需要调整总题数或条件。

**修正：** 共20道，做对得5分，做错扣2分。做了全部20题，得83分。
设做对x，做错20-x
5x-2(20-x)=83
5x-40+2x=83
7x=123
x≈17.57 ❌ 也不是整数。

**再改：** 20题，做对得5分，做错扣2分，做了全部20题，得了**86**分。
5x-2(20-x)=86
5x-40+2x=86
7x=126
x=18
做对18，做错2 ✅

### 第4题（修正版）

**一次口算比赛共20道题，做对一道得5分，做错一道扣2分，不做不得分也不扣分。**

小明一共做了18道题（2道没做），总共得了86分。

（1）他做对了几道？做错了几道？
（2）没做的题如果被小红做了，小红全做对了，小红比小明多多少分？
（3）做对的题平均用时8.5秒，做错平均6.2秒，总用时多少秒？

**解题思路：**
（1）做了18道，设对x道，错18-x道
5x - 2(18-x) = 86
5x - 36 + 2x = 86
7x = 122
x = 122÷7 ≈ 17.43 ❌ 还不是整数！

**彻底重新选题：**

**一次口算比赛共20道题，做对得5分，做错扣3分。**

小红做了全部20题，得了84分。

（1）做对几道？错几道？
（2）做对每题8.5秒，做错每题6.2秒，总用时多少秒？

**解题思路：**
（1）设对x，错20-x
5x - 3(20-x) = 84
5x - 60 + 3x = 84
8x = 144
x = 18 ✅
对18道，错2道 ✅

（2）总用时=18×8.5+2×6.2=153+12.4=165.4（秒）

**答案：** （1）对18道、错2道 （2）165.4秒

### 第5题（串联：小数意义 + 三角形内角和 + 运算律——期末模拟）

**本学期核心知识点"一锅端"——看看你掌握得怎么样：**

（1）0.68里面有（  ）个0.01，再添上（  ）个0.01就是1。

（2）一个等腰三角形的顶角是80°，它的一个底角是多少度？
   如果把它沿着对称轴剪开，得到两个完全一样的______三角形。

（3）简便计算：25 × 3.6 × 0.4 + 1.4 = ？

（4）小明在计算3.6×100时，算成了3.6×10，结果比正确结果少多少？

**解题思路：**
（1）0.68=68个0.01，1=100个0.01
还需要100-68=32个0.01。

（2）顶角80°，底角=(180-80)÷2=50°
两个底角各50°。
沿对称轴剪开（高线），得到两个直角三角形（有一个角是90°）。

（3）25×3.6×0.4+1.4
先算25×0.4×3.6=10×3.6=36（乘法交换律）
再加1.4=37.4 ✅

（4）3.6×100=360，3.6×10=36
差=360-36=324
或者：相当于少乘了90倍，差=3.6×90=324 ✅

**答案：** （1）68个、32个 （2）50°、直角 （3）37.4 （4）324
