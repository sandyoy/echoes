## 二、业务模式

圆爱康正在从传统模式向新模式转型，目前两种模式并存。

### 🔴 模式A：医院为主体，圆爱康为第三方

**逻辑：** 医院作为医疗服务主体，圆爱康提供技术与运营支持。

**钱流：**
```
患者 → 付钱给医院 → 医院扣管理费（45-60%）→ 剩余分给圆爱康
```

**问题：** 钱进了医院渠道后很难出来，圆爱康现金流紧张。且医院强势，往往要求50-60%的管理费分成。

---

### 🟢 模式B：圆爱康为服务主体，向医院购买医疗服务

**核心转变：** 圆爱康作为服务主体，打包居家康复服务包，患者直接付给圆爱康。服务包拆成两部分：

```
圆爱康（服务主体）→ 打包「居家康复服务包」
  ├── 🏥 医疗服务部分（圆爱康做不了）
  │    ├─ 医生远程开处方
  │    ├─ 医生远程制定/调整康复方案
  │    ├─ 医生解读评估报告
  │    └─ 远程诊疗相关医疗服务
  │    └──→ 圆爱康向医院购买这些医疗服务
  │
  └── 📦 非医疗服务部分（圆爱康自己做）
       ├─ 物联网设备提供+租赁
       ├─ 患者居家治疗全流程服务管理
       ├─ 线上客服（日常督促、使用指导）
       ├─ 设备物流+清洗消毒+再包装
       └─ 线下督导等

圆爱康（收钱方）→ 向医院支付医疗服务采购费（占服务包30-60%，视谈判）
```

**优势：**
- ✅ 钱不进医院渠道，圆爱康掌握现金流主动权
- ✅ 谈判地位平等——「我买你的服务」而非「你分给我钱」
- ✅ 国家已放开互联网+远程医疗服务自主定价，为采购模式提供了政策依据

**对标案例：** 腾讯健康向医院体检科购买体检服务的模式——表面买体检，实际是两个服务拼装在一起，各自占比成为谈判筹码。

**三种结算路径：**

| 方式 | 钱流 | 案例 |
|------|------|------|
| **方式A（分账模式）** | 患者→医院→医院扣管理费→分给圆爱康 | 早期项目 |
| **方式B（医院统一收款）** | 患者→医院公众号平台→次月按协议分账 | 部分早期项目 |
| **方式C（圆爱康平台收款）** ✅ | **患者→圆爱康平台→圆爱康分给医院** | **珠海妇幼、香洲人民医院、玉树妇幼** |

## 七、核心壁垒

### 「六师共管」服务体系

圆爱康独创的N对一服务体系，由六类专业人员组成：

| 角色 | 来源 | 职责 |
|------|------|------|
| 专科医生 | 医院 | 制定方案、解读评估、调整治疗计划 |
| 康复治疗师 | 医院 | 在线指导、评估、调整方案 |
| 护士 | 医院 | 患者宣教、培训设备操作 |
| 线上客服 | 圆爱康 | 非医疗事务、日常督促、使用指导 |
| 设备工程师 | 圆爱康 | 设备远程排查、故障处理 |
| 线下督导 | 圆爱康 | 线下指导、现场支持 |

### 「院内-院外」一体化闭环

- 患者数据在医院系统与居家设备间双向互联
- 医生可远程查看患者训练数据，实时调整方案
- 实现「院内治疗 → 居家康复 → 复诊评估」的完整闭环

### 零成本投入模式

- 医院无需资金投入（零设备采购、零平台开发、零人员增加）
- 圆爱康承担全部软硬件成本，从服务分成回收

### 完整运营闭环

从设备研发→平台开发→医院对接→患者培训→康复管理→设备回收消毒→再投放，全链条运营能力形成较高壁垒。

### 政策合规

对标「健康中国2030」、国办发〔2018〕26号《关于促进"互联网+医疗健康"发展的意见》等国家政策，等保三级认证，数据安全合规。
