## 一、公司概况

**公司名称**：圆爱康生物科技有限公司  
**品牌名称**：「智慧家康」  
**核心定位**：为医疗机构搭建 **"互联网+居家康复"综合运营服务平台**，连接医院与患者家庭

---

## 二、业务范围

圆爱康的业务模式是 **"医院主导、第三方赋能"** —— 医院作为医疗服务主体，圆爱康作为技术与运营支持方，提供全链条服务。

### 2.1 核心业务线

| 业务线 | 说明 |
|--------|------|
| **居家康复服务** | 为医院搭建居家医疗综合运营服务平台，患者在家完成康复治疗 |
| **院内智慧康复中心** | 为医院建设"智能化女性整体健康管理中心"，提供全套设备+系统+运营方案 |
| **CMS专科服务管理系统** | 盆底专科/康复科数字化管理系统，含患者管理、预约、设备管理等 |
| **物联网设备租赁** | 提供生物反馈电刺激仪等设备的租赁、物流、消毒、质检全流程 |
| **"六师共管"服务体系** | 专科医生+康复师+护士+客服+设备工程师+线下督导，N对一服务 |

### 2.2 服务对象

- **直接客户**：公立医院（三级甲等为主）的妇科、产科、康复科、泌尿科等科室
- **最终用户**：有周期性康复需求的患者（盆底康复、产后康复、骨科术后等）
- **渠道伙伴**：代理商（代理合同模式）

## 六、商业模式

### 6.1 合作模式：「医院主导、第三方赋能」

```
医院（医疗主体）
 ├─ 提供公众号/互联网医院入口
 ├─ 提供医疗技术支持（医生在线制定方案、调整方案）
 ├─ 收取治疗费/管理费（≥45%-52%）
 └─ 拥有全部医疗数据所有权

圆爱康（技术运营方）
 ├─ 提供软件平台（APP/小程序/管理端）
 ├─ 提供硬件设备（物联网康复设备，免费投入）
 ├─ 提供"六师共管"服务体系
 ├─ 负责物流、消毒、质检、耗材
 └─ 收取协同服务费（套餐剩余部分）
```

### 6.2 收费方式

**两种结算路径**：

| 方式 | 说明 |
|------|------|
| **方式A（分账模式）** | 医院治疗费在医院收取，圆爱康协同管理费在平台收取 |
| **方式B（统一收款）** | 套餐费用在医院公众号平台统一收取，次月按协议分账 |

**价格参考**：
- 患者居家康复套餐：约 **800-1200元/疗程**（协同服务费）
- 医院收取管理费比例：**45%-52%**
- 医院年度收益测算（以年门诊量100万为例）：约 **1000万元**

### 6.3 代理模式

- 代理商代理合同（费用收至圆爱康模式 / 费用收至医院指定平台模式）
- 推荐合作框架协议（介绍费3%）
- 续签代理商费率上浮15%

## 七、核心壁垒

### 7.1 「六师共管」服务体系

圆爱康独创的 **"六师共管"** N对一服务体系，由六类专业人员组成：

| 角色 | 来源 | 职责 |
|------|------|------|
| 专科医生 | 医院 | 制定方案、解读评估、调整治疗计划 |
| 康复治疗师 | 医院 | 在线指导、评估、调整方案 |
| 护士 | 医院 | 患者宣教、培训设备操作 |
| 线上客服 | 圆爱康 | 非医疗事务、日常督促、使用指导 |
| 设备工程师 | 圆爱康 | 设备远程排查、故障处理 |
| 线下督导 | 圆爱康 | 线下指导、现场支持 |

### 7.2 「院内-院外」一体化闭环

- 患者数据在医院系统与居家设备间**双向互联**
- 医生可**远程查看**患者训练数据，**实时调整**方案
- 实现了"院内治疗 → 居家康复 → 复诊评估"的**完整闭环**

### 7.3 零成本投入模式

- 医院**无需资金投入**（零设备采购、零平台开发、零人员增加）
- 圆爱康承担全部软硬件成本，从套餐分成回收
- 极大降低了医院的决策门槛

### 7.4 政策合规优势

严格对标国家政策体系：
- 《"健康中国2030"规划纲要》
- 国办发〔2018〕26号《关于促进"互联网+医疗健康"发展的意见》
- 国卫老龄发〔2024〕40号《关于促进医养结合服务高质量发展的指导意见》
- 广东省"互联网+医疗健康"行动计划（2023-2025）
- 互联网诊疗监管细则（试行）
- 等保三级认证，数据安全合规

### 7.5 完整的运营闭环

从**设备研发 → 平台开发 → 医院对接 → 患者培训 → 康复管理 → 设备回收消毒 → 再投放**的全链条运营能力，形成了较高的运营壁垒。
